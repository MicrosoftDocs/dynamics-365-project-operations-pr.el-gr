# <a name="create-a-work-breakdown-structure-wbs"></a>Δημιουργία μιας δομής ανάλυσης εργασίας (WBS)

Ένα χρονοδιάγραμμα έργου ενημερώνει για τις εργασίας που πρέπει να ολοκληρωθούν, τους πόρους που θα εκτελέσουν την εργασία, καθώς και το χρονικό πλαίσιο στο οποίο πρέπει να ολοκληρωθεί αυτή η εργασία. Το χρονοδιάγραμμα αντανακλά όλες τις εργασίες που σχετίζονται με την παράδοση του έργου εγκαίρως. Στις Dynamics 365 Project Operations, μπορείτε να δημιουργήσετε ένα χρονοδιάγραμμα έργου κατά:

  - Ανάλυση έργου σε εργασίες διαχείρισης.
  - Υπολογισμό του χρόνου που απαιτείται για την εκτέλεση κάθε εργασίας.
  - Ρύθμιση εξαρτήσεων εργασίας.
  - Ρύθμιση διάρκειας εργασίας.
  - Εκτίμηση των γενικών πόρων που θα εκτελέσουν τις εργασίες. 

Το χρονοδιάγραμμα έργου δημιουργείται στην καρτέλα **Χρονοδιάγραμμα** της σελίδας **Έργο**.

## <a name="tasks"></a>Εργασίες

Το πρώτο βήμα για τη δημιουργία ενός χρονοδιαγράμματος έργου είναι η διαίρεση του έργου σε διαχειρίσιμα τμήματα. Το χρονοδιάγραμμα στο Project Operations υποστηρίζει τις εξής δυνατότητες:

- Εργασίες σύνοψης ή κοντέινερ
- Εργασίες κόμβου φύλλου

### <a name="summary-tasks"></a>Σύνοψη εργασιών

Οι εργασίες σύνοψης μπορούν να αποθηκεύουν άλλες εργασίες σύνοψης ή εργασίες κόμβων φύλλου. Δεν έχουν καμία εργασιακή προσπάθεια ή κόστος δικό τους. Αντ' αυτού, η προσπάθεια και το κόστος εργασίας τους είναι μια συνάθροιση της προσπάθειας εργασίας και του κόστους των εργασιών κοντέινερ. Η ημερομηνία έναρξης της εργασίας σύνοψης είναι η ημερομηνία έναρξης των εργασιών κοντέινερ και η ημερομηνία λήξης είναι η ημερομηνία λήξης των εργασιών κοντέινερ. Είναι δυνατή η επεξεργασία του ονόματος μιας εργασίας σύνοψης, αλλά δεν είναι δυνατή η επεξεργασία των ιδιοτήτων χρονοδιαγράμματος (προσπάθεια, ημερομηνία και διάρκεια). Εάν διαγράψετε μια εργασία σύνοψης, διαγράφετε και όλες τις εργασίες κοντέινερ.

### <a name="leaf-node-tasks"></a>Εργασίες κόμβου φύλλου

Οι εργασίες κόμβου φύλλου αντιπροσωπεύουν τις πιο αναλυτικές εργασίες του έργου. Περιλαμβάνουν μια εκτιμώμενη προσπάθεια, πόρους, τις προγραμματισμένες ημερομηνίες έναρξης και λήξης και τη διάρκεια.

## <a name="create-a-task-hierarchy"></a>Δημιουργία ιεραρχίας εργασιών

### <a name="add-a-task"></a>Προσθήκη εργασίας

Ολοκληρώστε τα παρακάτω βήματα για να προσθέσετε μία ή περισσότερες εργασίες.

1. Μεταβείτε στα **Έργα** και επιλέξτε και ανοίξτε την καρτέλα έργου για το οποίο θέλετε να δημιουργήσετε ένα χρονοδιάγραμμα. 
2. Επιλέξτε την καρτέλα **Εργασίες**. 
3. Επιλέξτε **Προσθήκη νέας εργασίας**, καταχωρήστε ένα όνομα για την εργασία και, στη συνέχεια, πατήστε το πλήκτρο Enter.
2. Πληκτρολογήστε ένα άλλο όνομα εργασίας και πατήστε ξανά το πλήκτρο Enter μέχρι να έχετε μια πλήρη λίστα εργασιών.

### <a name="manage-hierarchy-of-a-task"></a>Διαχείριση ιεραρχίας μιας εργασίας

Όταν μια εργασία έχει εσοχή, γίνεται θυγατρικό στοιχείο της εργασίας που βρίσκεται ακριβώς πάνω από αυτήν. Στη συνέχεια, γίνεται υπολογισμός του αναγνωριστικού χρονοδιαγράμματος της εργασίας, ώστε να βασίζεται στο αναγνωριστικό χρονοδιαγράμματος της νέας γονικής της εργασίας και ακολουθεί το συνδυασμό αρίθμησης διάρθρωσης. Η γονική εργασία είναι πλέον μια εργασία σύνοψης. Ως εκ τούτου, γίνεται μια συνάθροιση των θυγατρικών εργασιών της. Όταν μια εργασία προωθείται, δεν είναι πλέον θυγατρική της εργασίας που ήταν η γονική της εργασία. Το αναγνωριστικό χρονοδιαγράμματος θα υπολογιστεί εκ νέου έτσι ώστε να αντικατοπτρίζει το ενημερωμένο βάθος και τη θέση της εργασίας στην ιεραρχία. Η προσπάθεια, το κόστος και οι ημερομηνίες της προηγούμενης γονικής εργασίας υπολογίζονται εκ νέου, ώστε να μην περιλαμβάνουν αυτήν την εργασία.

Ολοκληρώστε τα παρακάτω βήματα για να δημιουργήσετε εσοχή ή να προωθήσετε μια εργασία.

1. Στη σελίδα **Έργο**, στην καρτέλα **Εργασίες**, κάτω από τις **Εργασίες σύνοψης**, επιλέξτε τις τρεις κατακόρυφες κουκίδες με το όνομα της εργασίας και, στη συνέχεια, επιλέξτε **Δημιουργία δευτερεύουσας εργασίας**. 
2. Επιλέξτε την εργασία που θα δημιουργήσετε ή θα προωθήσετε. Για να επιλέξετε περισσότερες από μία εργασίες, επιλέξτε μια εργασία, πατήστε και κρατήστε πατημένο το πλήκτρο CTRL και, στη συνέχεια, επιλέξτε επιπλέον εργασίες.
2. Επιλέξτε **Εσοχή** ή **Προώθηση δευτερεύουσας εργασίας** για να μετακινήσετε εργασίες κάτω ή από τις εργασίες σύνοψης.

### <a name="move-tasks-up-and-down"></a>Μετακίνηση εργασιών προς τα επάνω και μετακίνηση εργασιών προς τα κάτω

Οι εργασίες μπορούν να μετακινηθούν σε οποιοδήποτε επίπεδο της δομής ανάλυσης εργασίας με έναν από τους εξής δύο τρόπους:

- Επιλέξτε μία ακόμα εργασία και σύρετέ την στη θέση που θέλετε.
- Επιλέξτε μία ή περισσότερες εργασίες, κάντε δεξί κλικ και επιλέξτε **Αποκοπή** επιλέξτε το κελί προορισμού του χρονοδιαγράμματος και, στη συνέχεια, κάντε δεξί κλικ και επιλέξτε **Επικόλληση**.

## <a name="task-attributes"></a>Χαρακτηριστικά εργασίας

Το όνομα μιας εργασίας περιγράφει την εργασία που πρέπει να ολοκληρωθεί. Στο Project Operations, τα χαρακτηριστικά που σχετίζονται με μια εργασία περιγράφουν το χρονοδιάγραμμα της εργασίας και τις απαιτήσεις στελέχωσης.

## <a name="schedule-attributes"></a>Χαρακτηριστικά χρονοδιαγράμματος

Τα χαρακτηριστικά **Προσπάθεια**, **Ημερομηνία έναρξης**, **Ημερομηνία λήξης** και **Διάρκεια** καθορίζουν το χρονοδιάγραμμα της εργασίας.

Ο παρακάτω πίνακας εμφανίζει πρόσθετα χαρακτηριστικά χρονοδιαγράμματος.

| **Τελικό εμφανιζόμενο όνομα** | **Τελική περιγραφή** |
| --- | --- |
| Η προσπάθεια ολοκληρώθηκε (ώρες) | Ολοκληρωμένο έργο για την εργασία σε ώρες. |
| Διάρκεια | Εμφανίζει τη διάρκεια της εργασίας σε ημέρες. |
| Συνολική προσπάθεια | Συνολικό έργο για την εργασία σε ώρες. |
| Τέλος | Ημερομηνία και ώρα λήξης. |
| % ολοκλήρωσης | Το ποσοστό της εργασίας που έχει ολοκληρωθεί. |
| Κάδος έργου | Ο πίνακας εργασιών μπορεί να ομαδοποιηθεί κατά σύνολο, έτσι ώστε κάθε σύνολο να έχει τη δική του στήλη. |
| Υπολειπόμενη προσπάθεια (ώρες) | Η υπολειπόμενη εργασία για την εργασία σε ώρες. |
| Έναρξη | Ημερομηνία και ώρα έναρξης. |
| Ονομασία | Το όνομα της εργασίας. |
| Αναγνωριστικό | Το αναγνωριστικό της εργασίας στη δομή ανάλυσης εργασίας. |

## <a name="staffing-attributes"></a>Χαρακτηριστικά προσωπικού

Τα χαρακτηριστικά στελέχωσης είναι προσβάσιμα μέσω του πεδίου **Πόροι** του χρονοδιαγράμματος. Μπορείτε είτε να αναζητήσετε έναν υπάρχοντα πόρο είτε να επιλέξετε **Δημιουργία** και στο παράθυρο **Γρήγορη δημιουργία** προσθέστε ένα μέλος της ομάδας έργου ως νέο πόρο.

Τα πεδία **Ρόλος**, **Μονάδα πόρων** και **Όνομα θέσης** χρησιμοποιούνται για την περιγραφή των απαιτήσεων στελέχωσης για την εργασία. Αυτά τα χαρακτηριστικά στελέχωσης, μαζί με το χρονοδιάγραμμα εργασιών, χρησιμοποιούνται για την εύρεση διαθέσιμων πόρων για την εκτέλεση αυτής της εργασίας.

   - **Ρόλος**: Καθορίζει τον τύπο του πόρου που απαιτείται για την εκτέλεση της εργασίας.
   - **Μονάδα πόρων**: Καθορίστε τη μονάδα από την οποία θα ανατίθενται οι πόροι για την εργασία. Αυτό το χαρακτηριστικό επηρεάζει το κόστος και την εκτίμηση πωλήσεων για την εργασία, εάν ο συντελεστής κόστους και χρέωσης για τον πόρο καθορίζεται με βάση τις μονάδες πόρων.
   - **Όνομα θέσης**: Καταγράψτε ένα φιλικό όνομα για τον γενικό πόρο, ο οποίος χρησιμεύει ως σύμβολο κράτησης θέσης για τον πόρο, ο οποίος θα κάνει τελικά την εργασία.

Το πεδίο **Πόροι** περιέχει το όνομα θέσης του γενικού πόρου ή του καθορισμένου πόρου όταν εντοπίζεται.

Το πεδίο **Κατηγορία** περιέχει τις τιμές που δηλώνουν έναν ευρύτερο τύπο εργασίας με τον οποίο η εργασία μπορεί να ομαδοποιηθεί. Αυτό το πεδίο δεν επηρεάζει τον προγραμματισμό ή την στελέχωση. Αντ' αυτού, το πεδίο χρησιμοποιείται μόνο για αναφορές.

## <a name="task-dependencies"></a>Εξαρτήσεις εργασίας

Μπορείτε να χρησιμοποιήσετε το χρονοδιάγραμμα στο Project Operations για τη δημιουργία προαπαιτούμενων σχέσεων μεταξύ εργασιών. Το πεδίο **Προκάτοχος** χρησιμοποιεί μία ή περισσότερες τιμές για να υποδείξει τις εργασίες από τις οποίες εξαρτάται μια εργασία. Όταν αντιστοιχίζετε τιμές προκατόχου σε μια εργασία, η εργασία μπορεί να ξεκινήσει μόνο όταν έχετε ολοκληρώσει όλες τις εργασίες του προκατόχου. Λόγω της εξάρτησης, η προγραμματισμένη ημερομηνία έναρξης της εργασίας επαναρυθμίζεται στην ημερομηνία κατά την οποία ολοκληρώνονται οι εργασίες προκατόχου.

Η λειτουργία εργασίας δεν επηρεάζει τις ενημερώσεις που πραγματοποιούνται στις ημερομηνίες έναρξης και λήξης των εργασιών που εξαρτώνται από τον προκάτοχό του.

## <a name="accessibility-and-keyboard-shortcuts"></a>Προσβασιμότητα και συντομεύσεις πληκτρολογίου

Το πλέγμα **Χρονοδιάγραμμα** είναι πλήρως προσβάσιμο και μπορεί να χρησιμοποιηθεί με προγράμματα ανάγνωσης οθόνης όπως Narrator, JAWS ή NVDA. Μπορείτε να μετακινηθείτε στην περιοχή πλέγματος χρησιμοποιώντας τα πλήκτρα βέλους (όπως στο Microsoft Excel), μπορείτε να χρησιμοποιήσετε το πλήκτρο Tab για να προωθήσετε τα αλληλεπιδραστικά στοιχεία του περιβάλλοντος εργασίας χρήστη και μπορείτε να χρησιμοποιήσετε το πλήκτρο με το κάτω βέλος, το πλήκτρο εισαγωγής (Enter) ή το πλήκτρο διαστήματος για να επιλέξετε και να ανοίξετε τα αναπτυσσόμενα μενού.
