# <a name="record-material-usage-on-projects-and-project-tasks"></a>Καταγραφή χρήσης υλικού σε έργα και εργασίες έργων

_**Ισχύει για:** Project Operations για σενάρια βασισμένα σε πόρους/μη εφοδιασμένα, ελαφριά ανάπτυξη - συμφωνία για προτιμολόγηση_

Καθώς μια ομάδα έργου εργάζεται με τις εργασίες σε ένα έργο, καταναλώνει ή χρησιμοποιεί υλικό. Ένα αρχείο καταγραφής χρήσης υλικού παρέχει έναν τρόπο καταγραφής αυτής της χρήσης, ώστε να μπορεί να εγκριθεί από τον υπεύθυνο έργου και τελικά να τιμολογηθεί στον πελάτη. 

Για να καταγράψετε τη χρήση του καταλόγου ή του εγγεγραμμένου υλικού και να το υποβάλετε στον υπεύθυνο έγκρισης, ακολουθήστε τα παρακάτω βήματα: 

1. Στο τμήμα παραθύρου περιήγησης, επιλέξτε **Χρήση υλικού** και, στη συνέχεια, επιλέξτε **Δημιουργία**.
2. Στη σελίδα **Νέα χρήση υλικού**, καταχωρήστε τις απαιτούμενες πληροφορίες χρήσης υλικού και, στη συνέχεια, επιλέξτε **Αποθήκευση**.

Ο παρακάτω πίνακας παρέχει πληροφορίες σχετικά με τα πεδία στη σελίδα **Αρχείο καταγραφής χρήσης υλικού**. 

| **Πεδίο** | **Περιγραφή** | **Κατάντη επίπτωση** |
| --- | --- | --- |
| Περιγραφή | Μια περιγραφή της συγκεκριμένης χρήσης του υλικού. | Δεν υπάρχει καμία κατάντη επίπτωση για αυτό το πεδίο. |
| Ημερομηνία | Την ημερομηνία κατά την οποία αναμένεται να χρησιμοποιηθεί το υλικό. | Δεν υπάρχει καμία κατάντη επίπτωση για αυτό το πεδίο. |
| Project | Έργα λίστας που είναι ενεργά. | Η επιλογή ενός έργου για ένα αρχείο καταγραφής χρήσης υλικού επηρεάζει το πεδίο **Εργασία** ώστε να εμφανίζονται μόνο οι εργασίες που υπάρχουν στο έργο. |
| Κλείσιμο εργασίας | Μια λίστα εργασιών για το έργο, συμπεριλαμβανομένων των εργασιών σύνοψης και κόμβων φύλλου. | Η επιλογή μιας εργασίας για ένα αρχείο καταγραφής χρήσης υλικού επηρεάζει το πραγματικό κόστος υλικού και τις πραγματικές πωλήσεις υλικού για μια εργασία. Εάν αυτό το πεδίο είναι κενό, το αντίστοιχο κόστος χρήσης υλικού και οι πωλήσεις παρακολουθούνται και συνοψίζονται μόνο στο επίπεδο έργου. |
| Επιλογή προϊόντος | Καθορίστε εάν η χρήση υλικού είναι για ένα **υπάρχον** προϊόν (καταλόγου) ή ένα **Εγγεγραμμένο** προϊόν. | Αυτό το πεδίο καθορίζει τον τύπο του προϊόντος. |
| Προϊόν | Το αναγνωριστικό του προϊόντος από κατάλογο προϊόντων. Όταν επιλέγετε ένα αναγνωριστικό προϊόντος, το πεδίο **Επιλογή προϊόντος** ενημερώνεται αυτόματα σε **Υπάρχον προϊόν**. Το αναγνωριστικό χρησιμοποιείται για την ανάκτηση των τιμών κόστους και πώλησης από τον τιμοκατάλογο. | Δεν υπάρχει καμία κατάντη επίπτωση για αυτό το πεδίο. |
| Περιγραφή εγγεγραμμένου προϊόντος | Ένα πεδίο κειμένου που θα γράψετε στο όνομα του προϊόντος. Αυτό το πεδίο είναι διαθέσιμο όταν επιλέγετε **Εγγεγραμμένο** προϊόν στο πεδίο **Επιλογή προϊόντος**.| Δεν υπάρχει καμία κατάντη επίπτωση για αυτό το πεδίο. |
| Πόρος με δυνατότητα κράτησης| Πόρος που χρησιμοποίησε αυτό το υλικό στο έργο. Η προεπιλογή για αυτό το πεδίο είναι ο πόρος με δυνατότητα κράτησης του συνδεδεμένου χρήστη, αλλά μπορεί να αλλάξει σε χρήση καρτέλας για λογαριασμό άλλων μελών της ομάδας έργου. | Δεν υπάρχει καμία κατάντη επίπτωση για αυτό το πεδίο. |
| Ομάδα μονάδων | Η προεπιλεγμένη τιμή σε αυτό το πεδίο προέρχεται από την ομάδα μονάδων που έχει οριστεί ως η προεπιλογή στο προϊόν καταλόγου. Μπορείτε να ενημερώσετε αυτό το πεδίο για να επιλέξετε μια άλλη ομάδα μονάδων. | Δεν υπάρχει καμία κατάντη επίπτωση για αυτό το πεδίο. |
| Μονάδα | Η προεπιλεγμένη τιμή σε αυτό το πεδίο είναι η προεπιλεγμένη μονάδα του επιλεγμένου προϊόντος. Μπορείτε να ενημερώσετε αυτό το πεδίο για να επιλέξετε μια άλλη μονάδα. | Η αλλαγή της μονάδας έχει ως αποτέλεσμα μια διαφορετική προεπιλεγμένη τιμή και κόστος μονάδας. |
| Ποσότητα | Η ποσότητα του προϊόντος που έχει χρησιμοποιηθεί στο έργο ή στην εργασία έργου. | Δεν υπάρχει καμία κατάντη επίπτωση για αυτό το πεδίο. |
| Κόστος μονάδας | Το κόστος μονάδας του επιλεγμένου προϊόντος και ο συνδυασμός μονάδων, όπως έχει οριστεί στον ισχύοντα τιμοκατάλογο κόστους. | Το κόστος μονάδας εμφανίζεται πάντα στη νομισματική μονάδα κόστους του έργου. Εάν δεν υπάρχει κόστος μονάδας για τον συνδυασμό προϊόντος και μονάδας στον τιμοκατάλογο, το κόστος μονάδας θα προοριστεί σε 0,00. |
| Συνολικό κόστος | Το ποσό κόστους που υπολογίζεται ως κόστος μονάδας \* ποσότητας.| Η ποσότητα κόστους εμφανίζεται πάντα στη νομισματική μονάδα κόστους του έργου. |


## <a name="submit-material-usage-for-review-and-approval"></a>Υποβολή χρήσης υλικού για αναθεώρηση και έγκριση 
Αφού καταγράψετε όλη τη χρήση του υλικού σας και είστε έτοιμοι να την εγκρίνετε, πρέπει να υποβάλετε τις πληροφορίες χρήσης για αναθεώρηση.

1. Μεταβείτε στο **Αρχείο καταγραφής χρήσης υλικού** και επιλέξτε μία ή περισσότερες καταχωρήσεις. Εναλλακτικά, επιλέξτε όλες τις καρτέλες χρήσης υλικού χρησιμοποιώντας το πλαίσιο ελέγχου στην κεφαλίδα.
2. Επιλέξτε **Υποβολή**. Το σύστημα επεξεργάζεται τις επιλεγμένες καταχωρήσεις και, στη συνέχεια, δημιουργεί αιτήσεις έγκρισης χρήσης υλικού.

## <a name="recall-a-material-usage-log"></a>Ανάκληση ενός αρχείου καταγραφής χρήσης υλικού

Όταν είναι απαραίτητο, μπορείτε να προσθέσετε μια χρήση υλικού που έχει υποβληθεί. Ο χρόνος που απαιτείται για την ανάκληση μιας εγγραφής χρήσης υλικού εξαρτάται από το στάδιο έγκρισης.  Εάν ο υπεύθυνος έγκρισης δεν έχει εγκρίνει ακόμα την εγγραφή, η ανάκληση μπορεί να πραγματοποιηθεί αμέσως. Ωστόσο, εάν η εγγραφή έχει ήδη εγκριθεί, ο υπεύθυνος έγκρισης καλείται να εγκρίνει την ανάκληση και να αντιστρέψει τις συναλλαγές.

1. Μεταβείτε στη **Χρήση υλικού** και στη λίστα με τα αρχεία καταγραφής χρήσης υλικού, επιλέξτε τη χρήση του υλικού που θα ανακληθεί.
2. Επιλέξτε **Ανάκληση**. Αν η εγγραφή χρήσης υλικού δεν έχει εγκριθεί ακόμα, το σύστημα θα την εγκρίνει αμέσως. Εάν η εγγραφή υλικού έχει ήδη εγκριθεί, δημιουργείται ένα αίτημα ανάκλησης για ειδοποίηση του υπεύθυνου έγκρισης ότι θέλετε να ανακαλέσετε τη χρήση του υλικού. Ο υπεύθυνος έγκρισης θα επιβεβαιώσει, στη συνέχεια, ότι μπορεί να γίνει η αντιστροφή και θα επιστραφεί η εγγραφή.

## <a name="delete-a-material-usage-log"></a>Διαγραφή ενός αρχείου καταγραφής χρήσης υλικού

Μπορείτε να διαγράψετε αρχεία καταγραφής χρήσης υλικού που δεν έχουν υποβληθεί. Για να διαγράψετε ένα αρχείο καταγραφής χρήσης υλικού που έχει ήδη υποβληθεί, πρέπει πρώτα να το ανακαλέσετε.
