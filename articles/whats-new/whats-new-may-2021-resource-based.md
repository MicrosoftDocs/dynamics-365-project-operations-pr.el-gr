# <a name="whats-new-may-2021---project-operations-for-resourcenon-stocked-based-scenarios"></a>Τι νέο υπάρχει τον Μάιο του 2021 - Project Operations για σενάρια πόρων/μη εφοδιασμένα σενάρια

_**Ισχύει για:** Project Operations για σενάρια βασισμένα σε πόρους/μη εφοδιασμένα_

Αυτό το θέμα ισχύει για τα ακόλουθα στοιχεία και τις εκδόσεις του Dynamics 365 Project Operations:

- Project Operations στην έκδοση περιβάλλοντος Dynamics 365 Dataverse 4.10.0.186
- Διαχείριση έργων και λογιστική σε περιβάλλοντα εφαρμογών Finance and Operations έκδοση 10.0.18

## <a name="features-included-in-this-release"></a>Δυνατότητες που περιλαμβάνονται σε αυτήν την έκδοση

Τα ακόλουθα χαρακτηριστικά περιλαμβάνονται σε αυτήν την έκδοση:

- [Λειτουργίες προγραμματισμού](../project-management/scheduling-modes.md): οι διαχειριστές έργου μπορούν να καθορίσουν εάν η διαχείριση ενός έργου πρέπει να γίνει με χρήση μιας σταθερής διάρκειας, σταθερής εργασίας ή λειτουργίας προγραμματισμού σταθερών μονάδων.
- [Ρύθμιση διανυθείσας απόστασς με χρήση επιπέδων ποσοστού διανυθείσας απόστασης](../expense/set-up-mileage.md): ενημέρωση επιπέδων ποσοστού διανυθείσας απόστασης για τις αναφορές δαπανών υπαλλήλων.

## <a name="project-operations-dual-write-maps-updates"></a>Ενημερώσεις χαρτών διπλής εγγραφής Project Operations

Στην παρακάτω λίστα εμφανίζονται οι αντιστοιχίσεις διπλής εγγραφής που έχουν τροποποιηθεί ή προστεθεί στην έκδοση Project Operations Μαΐου 2021.

| Αντιστοίχιση οντότητας | Ενημερωμένη έκδοση | Σχόλια |
| --- | --- | --- |
| Προέλευση χρηματοδότησης έργου (msdyn\_projectcontractsplitbillingrules) | 1.0.0.2 | Συγχρονισμός των όρων πληρωμής για τους πελάτες σύμβασης έργου. |
| Πρόταση τιμολογίου έργου V2 (τιμολόγια) | 1.0.0.3 | Συγχρονισμός όρων πληρωμής προτιμολογίου. |
| Οντότητα εξαγωγής γραμμής τιμολογίου πωλητή έργου ενοποίησης Project Operations (msdyn\_projectvendorinvoicelines) | 1.0.0.1 | Ενημερώσεις ποιότητας |
| Έργα V2 (msdyn\_projects) | 1.0.0.2 | Ενημερώσεις ποιότητας |

Να εκτελείτε πάντα η πιο πρόσφατη έκδοση του χάρτη στο περιβάλλον σας και να ενεργοποιείτε όλες τις σχετικές αντιστοιχίσεις πινάκων καθώς ενημερώνετε την έκδοση λύσης Project Operations Dataverse και έκδοση λύσης εφαρμογών Finance and Operations. Ορισμένες δυνατότητες και ικανότητες ενδεχομένως να μην λειτουργούν σωστά εάν η πιο πρόσφατη έκδοση του χάρτη δεν ενεργοποιηθεί. Μπορείτε να δείτε την ενεργή έκδοση της αντιστοίχισης στη στήλη **Έκδοση** στη σελίδα **Διπλή εγγραφή**. Για να ενεργοποιήσετε μια νέα έκδοση του χάρτη, επιλέξτε **Εκδόσεις χάρτη πίνακα**, επιλέξτε την πιο πρόσφατη έκδοση και, στη συνέχεια, αποθηκεύστε την επιλεγμένη έκδοση. Εάν έχετε προσαρμόσει έναν έτοιμο χάρτη πίνακα, αντιστοιχιστείτε ξανά τις αλλαγές. Για περισσότερες πληροφορίες, βλ. [Διαχείριση του κύκλου ζωής εφαρμογής](/dynamics365/fin-ops-core/dev-itpro/data-entities/dual-write/app-lifecycle-management.md).

Εάν αντιμετωπίσετε κάποιο πρόβλημα με την εκκίνηση του χάρτη, ακολουθήστε τις οδηγίες στο [ζήτημα των στηλών πίνακα που λείπουν στην ενότητα χάρτες](/dynamics365/fin-ops-core/dev-itpro/data-entities/dual-write/dual-write-troubleshooting-finops-upgrades.md#missing-table-columns-issue-on-maps) του οδηγού αντιμετώπισης προβλημάτων διπλής εγγραφής.
