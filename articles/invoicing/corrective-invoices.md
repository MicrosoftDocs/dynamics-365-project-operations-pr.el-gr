# <a name="create-corrective-project-based-invoices"></a>Δημιουργία διορθωτικών τιμολογίων βάσει έργου 

_**Ισχύει για:** Project Operations για σενάρια βασισμένα σε πόρους/μη εφοδιασμένα_

Ένα επιβεβαιωμένο τιμολόγιο έργου μπορεί να διορθωθεί για την επεξεργασία αλλαγών ή πιστώσεων σύμφωνα με τις διαπραγματεύσεις του πελάτη και του υπεύθυνου έργου.

Για να κάνετε αλλαγές σε ένα επιβεβαιωμένο τιμολόγιο, ανοίξτε το επιβεβαιωμένο τιμολόγιο και επιλέξτε **Διόρθωση αυτού του τιμολογίου**. 

> [!NOTE]
> Αυτή η επιλογή δεν είναι διαθέσιμη, εκτός εάν επιβεβαιωθεί ένα τιμολόγιο έργου.

Δημιουργείται ένα νέο προσχέδιο τιμολογίου από το επιβεβαιωμένο τιμολόγιο. Όλες οι λεπτομέρειες της γραμμής τιμολογίων από το τιμολόγιο που έχει ήδη επιβεβαιωθεί αντιγράφονται στο νέο προσχέδιο. Ακολουθούν ορισμένα βασικά σημεία που θα σας βοηθήσουν να κατανοήσετε περισσότερα σχετικά με τις λεπτομέρειες γραμμής στο νέο διορθωμένο τιμολόγιο:

- Όλες οι ποσότητες ενημερώνονται στο μηδέν. Αυτό προϋποθέτει ότι όλα τα στοιχεία που έχουν τιμολογηθεί έχουν πιστωθεί πλήρως. Εάν χρειάζεται, μπορείτε να ενημερώσετε με μη αυτόματο τρόπο αυτές τις ποσότητες ώστε να αντικατοπτρίζουν την ποσότητα που τιμολογείται και όχι την ποσότητα που πιστώνεται. Με βάση την ποσότητα που καταχωρείτε, η εφαρμογή υπολογίζει την πιστωθείσα ποσότητα. Αυτό το ποσό αντικατοπτρίζεται στις πραγματικές τιμές που δημιουργούνται όταν επιβεβαιώνεται το διορθωμένο τιμολόγιο. Εάν πραγματοποιείτε αλλαγές στο ποσό του φόρου, πρέπει να εισαγάγετε το σωστό ποσό φόρου και όχι το ποσό του φόρου που πιστώνεται.
- Οι διορθώσεις ορόσημων διεκπεραιώνονται πάντα ως πλήρεις πιστώσεις.
- Τα ποσά προκαταβολής μπορούν να διορθωθούν εάν ο πελάτης τιμολογηθεί για εσφαλμένο ποσό.
- Οι συμφωνίες προκαταβολών είναι δυνατό να διορθωθούν εάν χρησιμοποιήθηκε εσφαλμένο ποσό για τη συμφωνία σε σχέση με χρεώσεις σε ένα τιμολόγιο που έχει ήδη επιβεβαιωθεί.

> [!IMPORTANT]
> Οι λεπτομέρειες γραμμής τιμολογίου που είναι διορθώσεις σε άλλες ήδη τιμολογημένες χρεώσεις, έχουν ορίσει το πεδίο **Διόρθωση** σε **Ναι**. Τα τιμολόγια που έχουν διορθωμένες τις λεπτομέρειες της γραμμής τιμολογίου έχουν ένα πεδίο που ονομάζεται **Έχει διορθώσεις** που έχει οριστεί επίσης σε **Ναι**.
