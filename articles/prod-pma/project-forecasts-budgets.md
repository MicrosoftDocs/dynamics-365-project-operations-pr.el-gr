# <a name="project-forecasts-and-budgets"></a>Προβλέψεις και προϋπολογισμοί έργου

[!include [banner](../includes/banner.md)]

Υπάρχουν δύο τρόποι για να διαχειριστείτε και να ελέγξετε τα έργα σας: προβλέψεις έργου και προϋπολογισμοί έργου. 

Χρησιμοποιήστε την πρόβλεψη έργου εάν ο οργανισμός σας έχει μια λειτουργική προοπτική και αν εστιάζει στα έσοδα και το κόστος που προκύπτουν από συγκεκριμένες συναλλαγές. Αν ο οργανισμός σας επικεντρώνεται περισσότερο σε οικονομικά ποσά, μπορείτε να χρησιμοποιήσετε τον προϋπολογισμό έργου. 

Τόσο οι προβλέψεις έργου όσο και οι προϋπολογισμοί έργου χρησιμοποιούν μοντέλα πρόβλεψης για να κρατήσουν τα προβλεπόμενα ποσά συναλλαγής. 

Κάθε μέθοδος έχει τα πλεονεκτήματά της. Θα πρέπει να λάβετε υπόψη σας τα παρακάτω σημεία προτού επιλέξετε μια μέθοδο για τον οργανισμό σας.

|   Μέθοδος εκχώρησης       |           Πρόβλεψη έργου            |        Προϋπολογισμός έργου                           |
|---------------------------|------------------------------------------|----------------------------------------------------|
| **Ανάθεση περιόδου**     | δεν μπορείτε να εκχωρήσετε ρητά συναλλαγές σε μια οικονομική περίοδο. Αντ' αυτού, η πρόβλεψη και ο έλεγχος της πρόβλεψης βασίζονται στη διάρκεια ζωής του έργου. Επειδή οι προβλέψεις βασίζονται σε μια συγκεκριμένη ημερομηνία, θα πρέπει να συμπεράνετε την περίοδο από την ημερομηνία. | Μπορείτε να εκχωρήσετε συναλλαγές σε ένα ολόκληρο έργο ή σε μια οικονομική περίοδο. Εάν αναθέσετε σε μια περίοδο, μπορείτε να μεταφέρετε μη χρησιμοποιηθέντα ποσά προς τα εμπρός στην επόμενη οικονομική περίοδο. |
| **Προβολή συναλλαγών**  | Μπορείτε να προβάλετε τις συναλλαγές στις φόρμες πρόβλεψης, όπου μπορείτε να δείτε τις προβλέψεις για ολόκληρη την εταιρεία και για όλα τα έργα, ανεξάρτητα από την ιεραρχία. Για να εστιάσετε σε ένα συγκεκριμένο έργο, πρέπει να φιλτράρετε τα δεδομένα.                                       | Μπορείτε να προβάλετε προϋπολογισμένες συναλλαγές για μία μόνο ιεραρχική δομή έργου. Για αυτόν το λόγο, μπορείτε να προβάλετε τις λεπτομέρειες της συναλλαγής για ένα γονικό έργο ή για τα δευτερεύοντα έργα του.                 |
| **Μεταβλητές συναλλαγής** | Όταν καταχωρείτε συναλλαγές πρόβλεψης, μπορείτε να χρησιμοποιήσετε κάθε χαρακτηριστικό που υπάρχει για μια πραγματική συναλλαγή. Με αυτόν τον τρόπο μπορείτε να κάνετε περισσότερες λεπτομέρειες στις προβλέψεις. Για παράδειγμα, μπορείτε να καταχωρίσετε λεπτομέρειες για τις ιδιότητες "ποσότητες", "υπάλληλοι", "στοιχεία" ή "γραμμές".         | Όταν καταχωρείτε λεπτομέρειες προϋπολογισμού, μπορείτε να χρησιμοποιήσετε μόνο ποσά, κατηγορίες και δραστηριότητες.                    |
| **Ασφάλεια**              | Η πρόβλεψη βασίζεται στις συναλλαγές που καταχωρείτε στις φόρμες πρόβλεψης και δεν περιλαμβάνει μηχανισμό ελέγχου διεργασιών. Κάθε εργαζόμενος που έχει δικαιώματα για μια φόρμα πρόβλεψης μπορεί να αναθεωρήσει πληροφορίες χωρίς έγκριση.                                        | Ο προϋπολογισμός χρησιμοποιεί το σύστημα ροής εργασιών, το οποίο ενεργοποιεί τη διαχείριση αλλαγών και σας επιτρέπει να κρατήσετε ένα ιστορικό των αναθεωρήσεων.         |
| **Τύποι οντότητας**           | Οι εγγραφές συναλλαγών πρόβλεψης είναι βασισμένες στον αριθμό μονάδων και στις τιμές μονάδων κόστους και πωλήσεων.  | Τα στοιχεία του προϋπολογισμού υπολογίζονται με βάση τα ποσά, τα οποία διαιρούνται μεταξύ κόστους και εσόδων.                                          |
| **Μοντέλα πρόβλεψης**       | Επειδή κάθε πρόβλεψη πρέπει να συσχετιστεί με ένα μοντέλο, μπορείτε να δημιουργήσετε πολλαπλά μοντέλα προβλέψεων και επίσης να ρυθμίσετε υπομοντέλα.           | Ο προϋπολογισμός έργου περιορίζει τα μοντέλα πρόβλεψης που χρησιμοποιούνται για τον προϋπολογισμό. Λιγότερα μοντέλα πρόβλεψης μπορούν να συμβάλουν στη βελτίωση της συνέπειας στις προβολές.                           |
| **Υπερβάσεις κόστους**         | Μπορείτε να επιτρέψετε ή να απαγορεύσετε μόνο την είσοδο συναλλαγών που θα προκαλέσουν υπέρβαση του κόστους.   | Ο προϋπολογισμός έργου παρέχει πρόσθετες επιλογές ελέγχου για τους χρήστες. Μπορείτε να επιτρέψετε προειδοποιήσεις και υπερβάσεις.                    |
| **Στοιχείο ελέγχου**               | Το στοιχείο ελέγχου πρόβλεψης εκτελείται με τη χρήση της μείωσης πρόβλεψης. Τα πραγματικά ποσά αφαιρούνται από τα υπόλοιπα των συναλλαγών προβλέψεων χωρίς ίχνος ελέγχου. Αυτό μπορεί να καταστήσει δυσκολότερη την ανίχνευση του σημείου όπου πραγματοποιήθηκαν οι πραγματικές συναλλαγές.                   | Στο στοιχείο ελέγχου προϋπολογισμού έργου, τα πραγματικά ποσά αφαιρούνται από τα ποσά του υπόλοιπου προϋπολογισμού. Αυτό επιτρέπει ένα σαφέστερο ίχνος ελέγχου.                                   |

## <a name="project-forecasts"></a>Προβλέψεις έργου
Όταν χρησιμοποιείτε την πρόβλεψη έργου, μπορείτε να εισαγάγετε τις συναλλαγές πρόβλεψης σε φόρμες πρόβλεψης για κάθε τύπο συναλλαγής. Κάθε χαρακτηριστικό που είναι διαθέσιμο για μια πραγματική συναλλαγή μπορεί να χρησιμοποιηθεί για μια συναλλαγή πρόβλεψης — για παράδειγμα, κερδοφορία της σειράς, χαρακτηριστικά της ουράς, εργαζόμενοι ή περιγραφές. Επίσης, μπορείτε να προβάλετε τη διάρκεια μετά την οποία θα επιβαρυνθείτε με ένα κόστος που θα τιμολογήσετε σε έναν πελάτη. 

Οι συναλλαγές προβλέψεων έργου βασίζονται σε μονάδες και ποσά. 

Πρέπει να συσχετίσετε κάθε πρόβλεψη έργου με ένα μοντέλο πρόβλεψης. Όταν καταχωρείτε μια συναλλαγή πρόβλεψης, προτείνεται αυτόματα ένα μοντέλο πρόβλεψης. Το μοντέλο πρόβλεψης λειτουργεί ως κοντέινερ για τις συναλλαγές πρόβλεψης. 

Μπορείτε να ορίσετε τα μοντέλα πρόβλεψης ως υπομοντέλα για ένα μοντέλο. Με αυτόν τον τρόπο μπορείτε να προβλέψετε την περιοχή, την χρονική περίοδο ή το τμήμα. Μπορείτε να αντιγράψετε τις συναλλαγές πρόβλεψης σε ένα μοντέλο για να δημιουργήσετε ένα νέο μοντέλο και μπορείτε επίσης να μεταφέρετε τις συναλλαγές στη γενική λογιστική. Επειδή υπάρχει μια σχέση ένα προς ένα ανάμεσα σε μια πρόβλεψη και ένα μοντέλο, κάθε μοντέλο πρόβλεψης δημιουργεί έναν ξεχωριστό προϋπολογισμό για ένα έργο. 

Τα μοντέλα πρόβλεψης μπορούν να χρησιμοποιήσουν τη μείωση των προβλέψεων ως μηχανισμό ελέγχου για τα έργα. Στη μείωση πρόβλεψης, οι πραγματικές συναλλαγές μειώνουν τα υπόλοιπα των συναλλαγών πρόβλεψης. Ωστόσο, επειδή η μείωση των προβλέψεων εφαρμόζεται στο υψηλότερο έργο στην ιεραρχία, παρέχει μια περιορισμένη προβολή των αλλαγών στην πρόβλεψη. Για παράδειγμα, εάν ένας εργαζόμενος είναι συσχετισμένος με ένα δευτερεύον έργο, οι πραγματικές συναλλαγές για τον εργαζόμενο καταχωρούνται στο γονικό έργο. Αυτό μπορεί να δυσκολέψει την παρακολούθηση των αλλαγών, επειδή δεν μπορείτε εύκολα να καθορίσετε τη συναλλαγή στην οποία το δευτερεύον έργο προκάλεσε μια μείωση στο ποσό πρόβλεψης. Ως εκ τούτου, είναι καλή ιδέα να δημιουργήσετε ένα αντίγραφο του μοντέλου πρόβλεψης για τη μείωση των προβλέψεων για χρήση. Στη συνέχεια, μπορείτε να χρησιμοποιήσετε τις αναφορές για να προβάλετε τις αρχικές προβλέψεις. 

Μπορείτε να αναθεωρήσετε, να αντιγράψετε, να διαγράψετε ή να μεταφέρετε προβλέψεις έργου σε έναν προϋπολογισμό γενικής λογιστικής. Ωστόσο, δεν υπάρχει στοιχείο ελέγχου διεργασίας. Κάθε εργαζόμενος που έχει δικαιώματα για μια φόρμα πρόβλεψης μπορεί να αναθεωρήσει πληροφορίες χωρίς έλεγχο.

-   **Αναθεώρηση** – Μπορείτε να αναθεωρήσετε μια συναλλαγή πρόβλεψης στις ίδιες φόρμες όπου πραγματοποιήθηκαν οι αρχικές καταχωρήσεις.
-   **Αντιγραφή ή διαγραφή** – Όταν αντιγράφετε συναλλαγές πρόβλεψης, αντιγράφετε τις γραμμές συναλλαγής ενός μοντέλου πρόβλεψης σε ένα άλλο μοντέλο πρόβλεψης. Όταν διαγράφετε μια πρόβλεψη, διαγράφετε τις συναλλαγές πρόβλεψης από ένα μοντέλο πρόβλεψης. Για να περιορίσετε τις συναλλαγές πρόβλεψης που αντιγράφονται ή διαγράφονται, επιλέξτε συγκεκριμένους τύπους συναλλαγών και ημερομηνίες. Με αυτόν τον τρόπο μπορείτε να αντιγράψετε ή να διαγράψετε μόνο συγκεκριμένα τμήματα μιας πρόβλεψης.
-   **Μεταφορά** – Όταν μεταφέρετε μια πρόβλεψη έργου σε έναν προϋπολογισμό γενικής λογιστικής, μεταφέρετε τις συναλλαγές πρόβλεψης ενός μοντέλου πρόβλεψης σε έναν προϋπολογισμό γενικής λογιστικής. Μπορείτε να αντικαταστήσετε όλες τις συναλλαγές που είχαν μεταβιβαστεί στον προϋπολογισμό γενικής λογιστικής στις οποίες μεταφέρετε την πρόβλεψη του έργου σας.

## <a name="project-budgets"></a>Προϋπολογισμοί έργου
Ο προϋπολογισμός έργου είναι μια απλούστερη μέθοδος από την πρόβλεψη, παρόλο που ενσωματώνει τα μοντέλα πρόβλεψης. Χρησιμοποιεί μια φόρμα καταχώρησης για τις αρχικές λεπτομέρειες και τις αναθεωρήσεις του προϋπολογισμού και επιτρέπει προβολές που βασίζονται μόνο στο ποσό, την κατηγορία ή τη δραστηριότητα. 

Στον προϋπολογισμό έργου, όλοι οι αρχικοί προϋπολογισμοί και οι αναθεωρήσεις πρέπει να αποσταλούν σε μια ροή εργασιών έργου για έγκριση. Οι ροές εργασιών σάς παρέχουν αυξημένο έλεγχο στη διεργασία και δημιουργούν μια καρτέλα ιστορικού αλλαγών. 

Ο προϋπολογισμός έργου μοιάζει με τον προϋπολογισμό γενικής λογιστικής, αλλά είναι ταχύτερη και πιο εύκολη η ρύθμισή του. Πολλές από τις επιλογές του προϋπολογισμού γενικής λογιστικής, όπως οι ακολουθίες αριθμών ή η νομισματική μονάδα, δεν χρειάζεται να οριστούν ξεχωριστά για τα έργα.

Οι προϋπολογισμοί έργου συσχετίζονται αυτόματα με δύο μοντέλα πρόβλεψης, ένα για τον αρχικό προϋπολογισμό και ένα για τον υπολειπόμενο προϋπολογισμό. Ως εκ τούτου, οι αναφορές που βασίζονται σε μοντέλα πρόβλεψης μπορούν να χρησιμοποιήσουν δεδομένα προϋπολογισμού. Όταν ένα προϋπολογισμός έργου έχει δεσμευτεί, το σύστημα δημιουργεί συναλλαγές πρόβλεψης με βάση τα συσχετισμένα μοντέλα, τα οποία χρησιμοποιούνται για λόγους αναφοράς και ελέγχου.

## <a name="forecast-models"></a>Μοντέλα πρόβλεψης
Τα μοντέλα πρόβλεψης έχουν μια ιεραρχία ενός επιπέδου. Αυτό σημαίνει ότι μια πρόβλεψη έργου πρέπει να συσχετιστεί με ένα μοντέλο πρόβλεψης.

Εάν χρησιμοποιείτε την πρόβλεψη έργου, μπορείτε να αναγνωρίσετε μοντέλα ως υπομοντέλα. Στη συνέχεια μπορείτε να δημιουργήσετε προβλέψεις κατά τμήμα, χρονική περίοδο ή περιοχή. Για παράδειγμα, μπορείτε να δημιουργήσετε ένα μοντέλο πρόβλεψης για ένα έτος και, στη συνέχεια, να δημιουργήσετε υπομοντέλα για τις τοπικές προβλέψεις βορειοανατολικά, νοτιοανατολικά, βορειοδυτικά και νοτιοδυτικά που υποβάλουν οι τοπικοί φορείς. Επιλέγοντας διάφορες επιλογές στις διαθέσιμες αναφορές, μπορείτε να προβάλετε τις πληροφορίες κατά συνολική πρόβλεψη ή κατά υπομοντέλο.
