<span data-ttu-id="05ae3-104">Όταν εγκρίνετε έναν πωλητή να εργαστεί ως υπεργολάβος, ίσως θελήσετε να παρακρατήσετε την πληρωμή στον πωλητή, μέχρι να σας πληρώσει ο πελάτης σας για το έργο.</span><span class="sxs-lookup"><span data-stu-id="05ae3-104">When you approve a vendor to work as a subcontractor, you might want to withhold payment to the vendor until your customer pays you for the project.</span></span> <span data-ttu-id="05ae3-105">Για να υποστηρίξετε αυτό το σενάριο, μπορείτε να ορίσετε όρους πληρωμή κατά την πληρωμή (PWP), όταν ρυθμίζετε την παραγγελία αγοράς (PO) με τον προμηθευτή.</span><span class="sxs-lookup"><span data-stu-id="05ae3-105">To support this scenario, you can set up pay-when-paid (PWP) terms when you set up the purchase order (PO) with the vendor.</span></span>
