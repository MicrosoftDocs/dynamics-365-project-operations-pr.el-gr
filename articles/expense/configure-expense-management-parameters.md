# <a name="configure-expense-management-parameters"></a>Ρύθμιση παραμέτρων διαχείρισης εξόδων

_**Ισχύει για:** Project Operations για σενάρια βασισμένα σε πόρους/μη εφοδιασμένα_

Αυτό το θέμα περιγράφει τις παραμέτρους που ελέγχουν τη γενική συμπεριφορά στη διαχείριση εξόδων.

## <a name="general"></a>Γενικός

| Πεδίο                                                    | Περιγραφή |
|----------------------------------------------------------|-------------|
| Τυπικό ποσοστό εξόδων μετακίνησης                                 | Καταχωρίστε το ποσοστό επιστροφής για τα έξοδα μετακίνησης. Αυτό το ποσοστό πολλαπλασιάζεται με τη χιλιομετρική απόσταση που καταχωρείται για τη δαπάνη για τον υπολογισμό του ποσού που επιστρέφεται για τη δαπάνη ταξιδιού. |
| Επικύρωση σκοπού εξόδων                                 | Ενεργοποιήστε αυτήν την επιλογή για να περιορίσετε τους χρήστες σε ένα υπάρχον σύνολο τιμών που έχει ρυθμιστεί στο πεδίο **Σκοπός της αναφοράς εξόδων** όταν δημιουργούν αναφορές εξόδων. |
| Προσωπικές δαπάνες που καταβάλλονται από                                | Επιλέξτε ποιος είναι υπεύθυνος για την πληρωμή των ποσών συναλλαγής με πιστωτική κάρτα που κατηγοριοποιούνται ως προσωπικά. |
| Εμφάνιση όλης της αναφοράς εξόδων στο προηγούμενο επίπεδο διερεύνησης               | Κάντε αυτήν την επιλογή για να εμφανίσετε όλα τα έξοδα μιας αναφοράς εξόδων όταν προβάλλονται οι λεπτομέρειες του αρχικού εγγράφου για ένα συγκεκριμένο κουπόνι που δημιουργήθηκε όταν καταχωρήθηκε η αναφορά εξόδων. |
| Η προ-έγκριση του ταξιδιού είναι υποχρεωτική                 | Επιλέξτε αυτή την επιλογή για να απαιτείται η υποβολή και η έγκριση μιας αίτησης ταξιδίου, προκειμένου ένας εργαζόμενος να μπορεί να υποβάλει μια αναφορά εξόδων. |
| Να επιτρέπεται η επεξεργασία διανομών πριν από την καταχώρηση            | Επιλέξτε εάν είναι δυνατή η επεξεργασία των διανομών μιας αναφοράς εξόδων προτού καταχωρηθεί. |
| Αξιολόγηση πολιτικών διαχείρισης δαπανών                     | Επιλέξτε πότε αξιολογούνται τα έξοδα για να διαπιστώσετε αν έχει γίνει παραβίαση μιας πολιτικής δαπανών. Οι δαπάνες μπορούν να αξιολογηθούν για παραβιάσεις όταν η εγγραφή εξόδων καταχωρείται και αποθηκεύεται ή όταν η δαπάνη υποβάλλεται για έγκριση. Οι κανόνες πολιτικής για τις αποδείξεις που απαιτούνται θα αξιολογηθούν όταν αποθηκευτεί η γραμμή εξόδων. |
| Να επιτρέπονται οι γραμμές διεταιρικών εξόδων                         | Επιλέξτε εάν οι δαπάνες για άλλες νομικές οντότητες μπορούν να καταχωρηθούν σε μια αναφορά εξόδων. |
| Να επιτρέπεται η επεξεργασία της συναλλαγματικής ισοτιμίας για τα έξοδα της πιστωτικής κάρτας | Κάντε αυτήν την επιλογή για να επιτρέψετε στο χρήστη να επεξεργάζεται τη συναλλαγματική ισοτιμία για τα εισαγόμενα έξοδα πιστωτικής κάρτας. |
| Πεδία ιεραρχίας πολλών επιπέδων για εμφάνιση                  | Επιλέξτε ποια πεδία υπεύθυνου έγκρισης εμφανίζονται όταν ο τύπος ανάθεσης ροής έχει οριστεί σε ιεραρχία και η επιλογή **Ιεραρχία** έχει ρυθμιστεί για χρήση της ιεραρχίας έγκρισης, έγκριση δαπανών πολλών επιπέδων. Όταν χρησιμοποιείται η ιεραρχία έγκρισης πολλών επιπέδων για μια ροή εργασιών, τα επιλεγμένα πεδία θα εμφανίζονται στην αναφορά εξόδων και είναι δυνατή η επεξεργασία τους. |
| Εισαγωγή αριθμού πιστωτικής κάρτας υπαλλήλου                        | Επιλέξτε εάν είναι δυνατή η εισαγωγή και η αποθήκευση ενός αριθμού 15 χαρακτήρων ή 16 χαρακτήρων στο πεδίο **Αναγνωριστικό κάρτας** στη σελίδα **Πιστωτικές κάρτες** για έναν υπάλληλο. |
| Επικύρωση σκοπού αίτησης ταξιδίου                      | Ενεργοποιήστε αυτήν την επιλογή για να περιορίσετε τους χρήστες σε ένα υπάρχον σύνολο τιμών που έχει ρυθμιστεί στο πεδίο **Σκοπός της αναφοράς εξόδων** όταν δημιουργούν αιτήσεις ταξιδίων. |

## <a name="financial"></a>Ενδιάμεσοι χρηματοπιστωτικοί οργανισμοί

| Πεδίο                                                                                    | Περιγραφή |
|------------------------------------------------------------------------------------------|-------------|
| Όνομα καθημερινού ημερολογίου καθολικού                                                                | Επιλέξτε το όνομα του ημερολογίου καθολικού στο οποίο καταχωρούνται οι εγκεκριμένες αναφορές εξόδων. |
| Ενεργοποίηση της επιστροφής φόρων από τα έξοδα                                                        | Κάντε αυτήν την επιλογή για να ενεργοποιήσετε την επιστροφή φόρων δαπανών για επιλέξιμες δαπάνες. Αυτή η επιλογή δεν μπορεί να επιλεγεί εάν έχουν ενεργοποιηθεί οι κανόνες των ΗΠΑ περί φορολογίας πωλήσεων και η χρήση κανόνων φόρου. |
| Άμεση καταχώρηση προκαταβολής μετρητών                                                           | Κάντε αυτήν την επιλογή για να καταχωρήσετε μια εγκεκριμένη προκαταβολή μετρητών όταν ολοκληρωθεί η διαδικασία πληρωμής και μεταφοράς. Εάν αυτή η επιλογή δεν είναι ενεργοποιημένη, η διαδικασία πληρωμής και μεταφοράς δημιουργεί ένα μη καταχωρισμένο ημερολόγιο καθολικού. |
| Σωστή ημερομηνία λογιστικής κατά την καταχώρηση                                                   | Κάντε αυτήν την επιλογή για να ενημερώσετε τη λογιστική ημερομηνία κατά την οποία καταχωρούνται τα έξοδα, εάν η τρέχουσα περίοδος είναι σε αναμονή ή κλειστή. Η ημερομηνία λογιστικής θα οριστεί στην επόμενη ανοιχτή περίοδο. |
| Να επιτρέπεται η ομαδοποίηση συναλλαγών με βάση το λογαριασμό αντιστάθμισης που καθορίζεται στη μέθοδο πληρωμής       | Κάντε αυτήν την επιλογή για να συνοψίσετε τις συναλλαγές εξόδων για μια αναφορά εξόδων, με βάση το λογαριασμό αντιστάθμισης που καθορίζεται στη μέθοδο πληρωμής για τις δαπάνες. |
| Συμπεριλαμβάνεται φόρος                                                                             | Κάντε αυτήν την επιλογή για να συμπεριλάβετε το φόρο πωλήσεων στο ποσό δαπάνης από προεπιλογή. |
| Αποδέσμευση επιβαρυνόμενων χρηματικών ποσών στο κλείσιμο των αιτήσεων ταξιδίου                                      | Επιλέξτε αυτή την επιλογή για να αποδεσμεύσετε επιβαρυνόμενα χρηματικά ποσά όταν κλείσει μια εγκεκριμένη αίτηση ταξιδίου. |
| Να επιτρέπεται η υποβολή αίτησης ταξιδίου σχετικά με την υπέρβαση του προϋπολογισμού του μητρώου προϋπολογισμού και του προϋπολογισμού έργου | Κάντε αυτήν την επιλογή για να επιτρέψετε στους εργαζομένους να υποβάλλουν αιτήσεις ταξιδίων για έγκριση, παρόλο που υπερβαίνουν είτε τον επιτρεπόμενο προϋπολογισμό που ορίζεται στο αρχείο καταχωρήσεων προϋπολογισμού είτε τον επιτρεπόμενο προϋπολογισμό για ένα έργο. |
| Να επιτρέπεται η υποβολή αναφοράς δαπανών σχετικά με την υπέρβαση του προϋπολογισμού του μητρώου προϋπολογισμού και του προϋπολογισμού έργου     | Κάντε αυτήν την επιλογή για να επιτρέψετε στους εργαζομένους να υποβάλλουν αναφορές εξόδων για έγκριση παρόλο που υπερβαίνουν είτε τον επιτρεπόμενο προϋπολογισμό που ορίζεται στο αρχείο καταχωρήσεων προϋπολογισμού είτε τον επιτρεπόμενο προϋπολογισμό για ένα έργο. |
| Να επιτρέπεται η έγκριση αναφοράς δαπανών σχετικά με την υπέρβαση του προϋπολογισμού του μητρώου μόνο                 | Κάντε αυτήν την επιλογή για να επιτρέψετε στους υπεύθυνους έγκρισης να εγκρίνουν τις αναφορές εξόδων που υπερβαίνουν τον επιτρεπόμενο προϋπολογισμό που ορίζεται στο αρχείο καταχωρήσεων προϋπολογισμού. |

## <a name="per-diem"></a>Ημερήσια αποζημίωση

| Πεδίο                                 | Περιγραφή |
|---------------------------------------|-------------|
| Ελάχιστες ώρες για ημερήσιες αποζημιώσεις            | Εισαγάγετε τον προεπιλεγμένο ελάχιστο αριθμό ωρών που πρέπει να εργαστεί ένας εργαζόμενος σε μια ημέρα για να μπορεί να λάβει μια ημερήσια αποζημίωση για τα έξοδα που σχετίζονται με τα ταξίδια. Αυτή η τιμή χρησιμοποιείται ως προεπιλεγμένη τιμή μόνο για το πεδίο **Ελάχιστες ώρες** για τα επίπεδα ανά ποσοστό ημερήσιας αποζημίωσης. |
| Ποσοστό γεύματος                          | Εισαγάγετε το προεπιλεγμένο ποσοστό των ημερήσιων εξόδων για γεύματα που χρησιμοποιείται την πρώτη και την τελευταία ημέρα του ταξιδιωτικού κόστους όταν το πεδίο **Υπολογισμός μείωσης γεύματος κατά** έχει οριστεί σε **Τύπος γεύματος ανά ημέρα** ή **Αριθμός γευμάτων ανά ημέρα**. Η εργάσιμη ημέρα της πρώτης και της τελευταίας ημέρας μπορεί να είναι μικρότερη από μια τυπική εργάσιμη ημέρα. Ως εκ τούτου, το ποσό της ημερήσιας αποζημίωσης αυτές τις ημέρες μπορεί να διαφέρει από το τυπικό ποσό. Εάν το ποσοστό έχει οριστεί σε **0** (μηδέν), οι μειώσεις για την πρώτη και την τελευταία ημέρα θα είναι 0,00. |
| Ποσοστό ξενοδοχείου                         | Καταχωρίστε το προεπιλεγμένο ποσοστό της ημερήσιας αποζημίωσης για τα ξενοδοχεία που χρησιμοποιούνται την πρώτη και την τελευταία ημέρα της δαπάνης που σχετίζεται με τα ταξίδια. Η εργάσιμη ημέρα της πρώτης και της τελευταίας ημέρας μπορεί να είναι μικρότερη από μια τυπική εργάσιμη ημέρα. Ως εκ τούτου, το ποσό της ημερήσιας αποζημίωσης αυτές τις ημέρες μπορεί να διαφέρει από το τυπικό ποσό. Εάν το ποσοστό έχει οριστεί σε **0** (μηδέν), οι μειώσεις για την πρώτη και την τελευταία ημέρα θα είναι 0,00. |
| Άλλο ποσοστό                         | Καταχωρίστε το προεπιλεγμένο ποσοστό της ημερήσιας αποζημίωσης για διάφορα έξοδα που χρησιμοποιούνται την πρώτη και την τελευταία ημέρα της δαπάνης που σχετίζεται με τα ταξίδια. Η εργάσιμη ημέρα της πρώτης και της τελευταίας ημέρας μπορεί να είναι μικρότερη από μια τυπική εργάσιμη ημέρα. Ως εκ τούτου, το ποσό της ημερήσιας αποζημίωσης αυτές τις ημέρες μπορεί να διαφέρει από το τυπικό ποσό. Εάν το ποσοστό έχει οριστεί σε **0** (μηδέν), οι μειώσεις για την πρώτη και την τελευταία ημέρα θα είναι 0,00. |
| Μείωση ποσοστού για πρωινό | Καταχωρίστε το ποσό μείωσης της ημερήσιας αποζημίωσης για το πρωινό. Για παράδειγμα, εάν ένας εργαζόμενος λάβει ένα δωρεάν πρωινό, μπορεί να θέλετε να μειώσετε το ποσό της ημερήσιας αποζημίωσης κατά 10 τοις εκατό. |
| Μείωση ποσοστού για μεσημεριανό     | Καταχωρίστε το ποσό μείωσης της ημερήσιας αποζημίωσης για το μεσημεριανό. Για παράδειγμα, εάν ένας εργαζόμενος λάβει ένα δωρεάν μεσημεριανό, μπορεί να θέλετε να μειώσετε το ποσό της ημερήσιας αποζημίωσης κατά 15 τοις εκατό. |
| Μείωση ποσοστού για δείπνο    | Καταχωρίστε το ποσό μείωσης της ημερήσιας αποζημίωσης για το δείπνο. Για παράδειγμα, εάν ένας εργαζόμενος λάβει ένα δωρεάν δείπνο, μπορεί να θέλετε να μειώσετε το ποσό της ημερήσιας αποζημίωσης κατά 25 τοις εκατό. |
| Υπολογισμός μείωσης γεύματος κατά           | Καθορίστε τον τρόπο με τον οποίο το σύστημα θα πρέπει να υπολογίζει τη μείωση της ημερήσιας αποζημίωσης γευμάτων επιλέγοντας εάν η μείωση θα είναι βασισμένη στον τύπο γεύματος ανά ταξίδι ή ανά ημέρα, ή εάν θα γίνεται με βάση τον αριθμό των γευμάτων που επιτρέπονται ανά ημέρα. |
| Στρογγυλοποίηση ημερήσιας αποζημίωσης                     | Επιλέξτε τον τύπο στρογγυλοποίησης που χρησιμοποιείται για δαπάνες ημερήσιας αποζημίωσης. Εάν επιλέξετε κανονική στρογγυλοποίηση, κάθε δαπάνη ημερήσιας αποζημίωσης που έχει ποσό 0,50 στρογγυλοποιείται προς τα επάνω σε 1,00 και κάθε δαπάνη ημερήσιας αποζημίωσης που έχει ποσό μικρότερο από 0,50 στρογγυλοποιείται προς τα κάτω στο 0,00. |
| Βάση υπολογισμού ημερήσιας αποζημίωσης          | Επιλέξτε αν μια ημερήσια αποζημίωση θα βασίζεται σε μια ημερολογιακή ημέρα ή σε μια περίοδο 24 ωρών. |

## <a name="fax-cover-pages"></a>Συνοδευτικές σελίδες φαξ

| Πεδίο                          | Περιγραφή |
|--------------------------------|-------------|
| Οδηγίες                   | Καταχωρίστε τις οδηγίες που πρέπει να ακολουθούν οι υπάλληλοι όταν δημιουργούν μια συνοδευτική σελίδα για ένα φαξ που χρησιμοποιείται για την αποστολή αποδείξεων που σχετίζονται με μια αναφορά εξόδων. Για να εισάγετε κείμενο που θα εμφανίζεται ανάλογα με τη γλώσσα, με βάση τη γλώσσα χρήστη, επιλέξτε **Μεταφράσεις**. |
| Αναγνωριστικό χρήστη (πληροφορίες γραμμικού κώδικα) | Κάντε αυτήν την επιλογή για να αποθηκεύσετε το μοναδικό αναγνωριστικό ενός υπαλλήλου στον γραμμικό κώδικα που χρησιμοποιείται στη συνοδευτική σελίδα του φαξ. |
| Γραμμικός κώδικας                       | Επιλέξτε τον γραμμικό κώδικα που χρησιμοποιείται στη συνοδευτική σελίδα του φαξ. Οι γραμμικοί κώδικες 39 και 128 υποστηρίζονται στη διαχείριση εξόδων. |

## <a name="anti-corruption"></a>Καταπολέμηση της διαφθοράς

| Πεδίο                                 | Περιγραφή |
|---------------------------------------|-------------|
| Εμφάνιση βεβαίωσης κατά της διαφθοράς   | Κάντε αυτήν την επιλογή για να εμφανίσετε το κείμενο κατά της διαφθοράς όταν δημιουργείται μια αναφορά εξόδων. Στη συνέχεια, μπορείτε να ενεργοποιήσετε συγκεκριμένες κατηγορίες εξόδων που θα απαιτούν την επιλογή της βεβαίωσης κατά της διαφθοράς στην αναφορά εξόδων. Για παράδειγμα, μια κατηγορία δώρου που σχετίζεται με μια κυβερνητική επίσημη δαπάνη μπορεί να απαιτήσει από τον υπάλληλο να επιβεβαιώσει ότι η δαπάνη ικανοποιεί την πολιτική της εταιρείας που σχετίζεται με κρατικούς αξιωματούχους. |
| Μήνυμα κατά της διαφθοράς για το άτομο που υποβάλλει | Καταχωρίστε το κείμενο που θα πρέπει να εμφανίζεται σε έναν υπάλληλο που δημιουργεί μια αναφορά εξόδων. Για να εισάγετε κείμενο που θα εμφανίζεται ανάλογα με τη γλώσσα, με βάση τη γλώσσα χρήστη, επιλέξτε **Μεταφράσεις**. |
| Μήνυμα κατά της διαφθοράς για το άτομο που εγκρίνει  | Καταχωρίστε το κείμενο που θα πρέπει να εμφανίζεται στο άτομο που υποβάλλει όταν δημιουργείται μια αναφορά εξόδων. Για να εισάγετε κείμενο που θα εμφανίζεται ανάλογα με τη γλώσσα, με βάση τη γλώσσα χρήστη, επιλέξτε **Μεταφράσεις**. |
