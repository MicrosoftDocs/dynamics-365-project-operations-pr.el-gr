# <a name="configure-expense-management"></a>Ρύθμιση παραμέτρων διαχείρισης εξόδων

Σε αυτό το άρθρο περιγράφονται οι σκέψεις και οι αποφάσεις που πρέπει να κάνετε κατά τη διαδικασία σχεδιασμού για να ρυθμίσετε τις παραμέτρους της διαχείρισης εξόδων. Στη διαχείριση εξόδων, μπορείτε να αποθηκεύσετε πληροφορίες σχετικά με μεθόδους πληρωμής, αιτήσεις για ταξίδια, αναφορές εξόδων, πολιτικές και ούτω καθεξής.

Επειδή πολλές από τις αποφάσεις που λαμβάνετε όταν προγραμματίζετε τη ρύθμιση παραμέτρων για τη διαχείριση εξόδων βασίζονται στην ιεραρχία και την οικονομική δομή του οργανισμού σας, πρέπει να ανατρέξετε στα έγγραφα σχεδιασμού για αυτές τις περιοχές.

## <a name="intercompany-expenses"></a>Διεταιρικές δαπάνες

Όταν ενεργοποιείτε τις διεταιρικές δαπάνες, επιτρέπετε σε νομικές οντότητες και εργαζομένους να επιβαρύνονται με έξοδα εκ μέρους άλλης νομικής οντότητας και να εισπράττουν πληρωμές από τη νομική οντότητα απασχόλησης εντός του οργανισμού σας. Για παράδειγμα, ένας εργαζόμενος στη νομική οντότητα Α ολοκληρώνει ένα έργο για τη νομική οντότητα Β και το έργο επιβαρύνεται με έξοδα ταξιδίου. Εάν τα διεταιρικά έξοδα είναι ενεργοποιημένα, ο εργαζόμενος μπορεί να καταχωρήσει μια αναφορά εξόδων που θα κοινοποιήσει τη δαπάνη στη νομική οντότητα Β και, στη συνέχεια, η δαπάνη θα πρέπει να καταβληθεί από την νομική οντότητα Α. Εάν ο οργανισμός σας δεν διαθέτει πολλές νομικές οντότητες, δεν χρειάζεται να ενεργοποιήσετε τις διεταιρικές δαπάνες.

**Απόφαση:** Θέλετε να ενεργοποιήσετε τις διεταιρικές δαπάνες;

## <a name="financial-management"></a>Χρηματοοικονομική διαχείριση

Η διαχείριση δαπανών είναι άρρηκτα ενσωματωμένη με τη χρηματοοικονομική διαχείριση του οργανισμού σας. Σε μεγάλο βαθμό, η ρύθμιση των παραμέτρων για τη διαχείριση εξόδων θα βασίζεται στις αποφάσεις που έχετε λάβει σχετικά με τα οικονομικά του οργανισμού σας. Οι παρακάτω ενότητες περιγράφουν τις διάφορες περιοχές που απαιτούν προγραμματισμό και αποφάσεις, βάσει των οικονομικών αποφάσεων και της καθοδήγησης του οργανισμού σας από την ηγετική σας ομάδα.

### <a name="per-diems"></a>Ημερήσιες αποζημιώσεις

Πρέπει να καθορίσετε τις ημερήσιες αποζημιώσεις εργαζόμενου που παρέχει ο οργανισμός σας. Επειδή οι ημερήσιες αποζημιώσεις χρησιμοποιούνται συνήθως για την κάλυψη εξόδων όπως τα γεύματα, η στέγαση και άλλα παρεπόμενα έξοδα, μπορείτε να δημιουργήσετε κανόνες για τις ημερήσιες αποζημιώσεις που προσφέρει ο οργανισμός σας. Οι τιμές ημερήσιας αποζημίωσης βασίζονται στην εποχή του χρόνου, στην τοποθεσία ταξιδίου ή και στα δύο. Όταν καθορίζετε έναν κανόνα ημερήσιας αποζημίωσης, μπορείτε να προσδιορίσετε ότι ένα ποσοστό της ημερήσιας αποζημίωσης θα παρακρατηθεί εάν ο εργαζόμενος λάβει δωρεάν γεύματα ή υπηρεσίες. Μπορείτε, επίσης, να ορίσετε τα επίπεδα ανά ποσοστό ημερήσιας αποζημίωσης για να ορίσετε τον ελάχιστο και τον μέγιστο αριθμό ωρών όπου θα ισχύει η τιμή της ημερήσιας αποζημίωσης για τα ταξίδια του εργαζομένου.

**Αποφάσεις:**

- Προεπιλεγμένοι κανόνες ημερήσιας αποζημίωσης για την πρώτη και την τελευταία ημέρα:

    - Ποιος είναι ο ελάχιστος αριθμός ωρών που μπορεί να διεκδικήσει ένας εργαζόμενος για μια ημέρα και να εξακολουθεί να λαμβάνει ημερήσια αποζημίωση;
    - Υπάρχει μείωση του ποσού που προσφέρεται για τα γεύματα για την πρώτη ημέρα και την τελευταία ημέρα; Σε περίπτωση μείωσης, ποιο είναι το ποσοστό της μείωσης;
    - Υπάρχει μείωση του ποσού που προσφέρεται από ένα ξενοδοχείο για την πρώτη ημέρα και την τελευταία ημέρα; Σε περίπτωση μείωσης, ποιο είναι το ποσοστό της μείωσης;
    - Υπάρχει μείωση του ποσού που προσφέρεται για άλλα έξοδα που προκύπτουν την πρώτη ημέρα και την τελευταία ημέρα; Σε περίπτωση μείωσης, ποιο είναι το ποσοστό της μείωσης;

- Προεπιλεγμένοι κανόνες ανά ημερήσια αποζημίωση:

    - Υπάρχει ποσοστό μείωσης της ημερήσιας αποζημίωσης για κάθε γεύμα, εάν, για παράδειγμα, το γεύμα είναι δωρεάν; Σε περίπτωση μείωσης, ποιο είναι το ποσοστό της μείωσης για κάθε γεύμα;
    - Η μείωση του γεύματος υπολογίζεται ανά ημέρα, ανά ταξίδι ή με βάση τον αριθμό των γευμάτων ανά ημέρα;
    - Οι ημερήσιες ποσότητες πρέπει να στρογγυλοποιούνται με τον κανονικό τρόπο ή να στρογγυλοποιούνται προς τα πάνω;
    - Οι ημερήσιες αποζημιώσεις υπολογίζονται σε μια διάρκεια 24 ωρών ή σε μια ημερολογιακή ημέρα;

- Οι κανόνες ημερήσιας αποζημίωσης βασίζονται στην τοποθεσία:

    - Οι ημερήσιες αποζημιώσεις ποικίλλουν ανάλογα με την τοποθεσία; Ποιες τοποθεσίες περιλαμβάνονται;
    - Εάν οι ημερήσιες αποζημιώσεις ποικίλλουν ανάλογα με την τοποθεσία, για κάθε τοποθεσία, ποιο είναι το ποσοστό που παρέχεται για τους ακόλουθους τύπους δαπανών:

        - Γεύματα
        - Ξενοδοχείο
        - Άλλα έξοδα

### <a name="expense-management-journals-and-accounts"></a>Ημερολόγια διαχείρισης εξόδων και λογαριασμοί

Η διαχείριση εξόδων απαιτεί τη χρήση πολλών ημερολογίων και λογαριασμών. Πρέπει να αποφασίσετε, για παράδειγμα, εάν ο ίδιος λογαριασμός χρησιμοποιείται για προκαταβολές μετρητών και διενέξεις πιστωτικών καρτών.

**Αποφάσεις:**

- Σε ποιο ημερολόγιο καθολικού καταχωρίζονται οι αναφορές εγκεκριμένων δαπανών;
- Ποιος λογαριασμός χρησιμοποιείται για προκαταβολές μετρητών;
- Θα πρέπει να καταχωρηθούν αμέσως προκαταβολές μετρητών;

### <a name="payment-methods"></a>Μέθοδοι πληρωμής

Όταν επιτρέπετε στους εργαζομένους να επιβαρύνονται με έξοδα εκ μέρους της επιχείρησής σας, πρέπει να ορίσετε τις μεθόδους πληρωμής που επιτρέπεται να χρησιμοποιούν οι υπάλληλοι. Για παράδειγμα, μπορείτε να επιτρέψετε στους εργαζομένους να χρησιμοποιούν μετρητά ή μια εταιρική πιστωτική κάρτα. Επίσης, θα μπορούσατε να επιτρέψετε στους εργαζομένους να χρησιμοποιούν προσωπικές πιστωτικές κάρτες και, στη συνέχεια, να αποζημιώνετε τους εργαζομένους. Πρέπει να λάβετε τις παρακάτω αποφάσεις για κάθε μέθοδο πληρωμής που θα επιτρέψετε.

**Αποφάσεις:**

- Ποιες μέθοδοι πληρωμής επιτρέπονται;
- Σε ποιον ανήκουν οι δαπάνες μεθόδου πληρωμής;
- Υπάρχει τύπος λογαριασμού αντιστάθμισης; Εάν υπάρχει τύπος λογαριασμού αντιστάθμισης, τι είναι;
- Εάν υπάρχει τύπος λογαριασμού αντιστάθμισης, ποιος είναι ο λογαριασμός;
- Εάν ο τρόπος πληρωμής είναι μια πιστωτική κάρτα, θα πρέπει να χρησιμοποιείται η μέθοδος πληρωμής μόνο με εισαγόμενες συναλλαγές;

### <a name="expense-categories-and-shared-categories"></a>Κατηγορίες εξόδων και κοινόχρηστες κατηγορίες

Όταν οι υπάλληλοι δημιουργούν μια αναφορά εξόδων, κάθε δαπάνη την οποία καταγράφουν πρέπει να συσχετίζεται με μια κατηγορία εξόδων. Οι κατηγορίες δαπανών προέρχονται από κοινόχρηστες κατηγορίες, οι οποίες είναι δυνατό να χρησιμοποιηθούν από κοινού σε όλες τις νομικές οντότητες του οργανισμού σας. Αυτές οι κατηγορίες είναι επίσης δυνατό να χρησιμοποιηθούν από κοινού στη διαχείριση έργου και στη λογιστική, ανάλογα με τον τρόπο με τον οποίο καθορίζεται ο οργανισμός σας. Με βάση τον ορισμό του οργανισμού σας και την καθοδήγησή σας από την ομάδα υλοποίησης, πρέπει να καθορίσετε εάν οι κατηγορίες που χρησιμοποιούνται στη διαχείριση δαπανών θα χρησιμοποιηθούν μόνο στη διαχείριση εξόδων ή θα πρέπει να είναι κοινόχρηστες μεταξύ της διαχείρισης έργων και λογιστικής και της διαχείρισης δαπανών. Λάβετε υπόψη ότι αυτές οι κατηγορίες είναι δυνατό να είναι κοινόχρηστες μεταξύ του έργου και των εξόδων ή του έργου και της παραγωγής αλλά όχι μεταξύ των εξόδων και της παραγωγής. Πρέπει να λάβετε τις παρακάτω αποφάσεις για κάθε κατηγορία εξόδων.

**Αποφάσεις:**

- Ποια είναι η κατηγορία εξόδων; Τα παραδείγματα περιλαμβάνουν κατηγορίες για πτήσεις, ξενοδοχεία ή χιλιόμετρα.
- Μπορεί η κατηγορία εξόδων να χρησιμοποιηθεί και στη διαχείριση έργου και στη λογιστική;
- Ποιος είναι ο τύπος εξόδων;
- Ποια είναι η προεπιλεγμένη μέθοδος πληρωμής για την κατηγορία εξόδων;
- Οι δαπάνες στην κατηγορία εξόδων πρέπει να είναι αναλυτικές;
- Ποιος είναι ο κύριος προεπιλεγμένος λογαριασμός για την κατηγορία εξόδων;
- Ποια είναι η προεπιλεγμένη ομάδα φόρου πωλήσεων είδους για την κατηγορία εξόδων;
- Επιτρέπονται επιπλέον μέθοδοι πληρωμής για την κατηγορία εξόδων; Εάν επιτρέπονται επιπλέον μέθοδοι πληρωμής, ποιες είναι;
- Υπάρχουν υποκατηγορίες σε αυτήν την κατηγορία εξόδων; Αν υπάρχουν υποκατηγορίες, πρέπει να έχετε αποφασίσει για τα εξής:

    - Αποκλείεται καμία από τις υποκατηγορίες από την είσπραξη φόρων;
    - Ποια είναι η ομάδα φόρου πωλήσεων ειδών στις υποκατηγορίες;

Εάν η κατηγορία εξόδων χρησιμοποιείται επίσης στη διαχείριση έργου και στη λογιστική, απαντήστε στις υπόλοιπες ερωτήσεις. Διαφορετικά, μεταβείτε στην επόμενη ενότητα.

- Ποιοι λογαριασμοί κόστους θα χρησιμοποιηθούν για τις ακόλουθες δαπάνες;

    - Κόστος
    - Εκχώρηση μισθοδοσίας
    - WIP-τιμή κόστους
    - Κόστος-στοιχείο
    - WIP-τιμή κόστους-στοιχείο
    - Συσσωρευμένη απώλεια
    - WIP-συσσωρευμένη απώλεια

- Ποιοι λογαριασμοί εσόδων θα χρησιμοποιηθούν για τα ακόλουθα;

    - Τιμολογημένα έσοδα
    - Συσσωρευμένα έσοδα - τιμή πωλήσεων
    - WIP-τιμή πωλήσεων
    - Συσσωρευμένα έσοδα-παραγωγή
    - WIP-παραγωγή
    - Συσσωρευμένα έσοδα-κέρδος
    - WIP-κέρδος
    - Συσσωρευμένα έσοδα-συνδρομή
    - WIP-συνδρομή

### <a name="taxes"></a>Φόροι

Για τους φόρους που σχετίζονται με τα έξοδα, πρέπει να καθορίσετε τι περιλαμβάνεται ή τι είναι ενεργοποιημένο στις αναφορές εξόδων.

**Αποφάσεις:**

- Ο φόρος πωλήσεων περιλαμβάνεται στα ποσά εξόδων;
- Θα πρέπει να ενεργοποιηθεί η είσπραξη φόρων στις δαπάνες;

    > [!NOTE]
    > Όταν σχεδιάζετε τη γενική λογιστική, εάν αποφασίσατε να εφαρμόσετε φόρο πωλήσεων στις ΗΠΑ και να χρησιμοποιήσετε φορολογικούς κανόνες, δεν μπορείτε να ενεργοποιήσετε την είσπραξη φόρων στις δαπάνες. (Για την εφαρμογή του φόρου πωλήσεων στις ΗΠΑ και τη χρήση φορολογικών κανόνων, ορίστε την επιλογή **Εφαρμογή κανόνων φορολόγησης φόρων πωλήσεων** σε **Ναι**.)

## <a name="policies"></a>Πολιτικές

Δημιουργώντας πολιτικές αναφοράς δαπανών, μπορείτε να βοηθήσετε τον οργανισμό σας να εξοικονομήσει χρόνο και χρήμα, όταν οι υπάλληλοι επιβαρύνονται με έξοδα για λογαριασμό του. Οι πολιτικές εγγυώνται ότι οι υπάλληλοι παραμένουν εντός προϋπολογισμού, παρέχουν όλες τις απαιτούμενες πληροφορίες και δαπανούν χρήματα μόνο εφόσον απαιτείται. Πρέπει να λάβετε τις παρακάτω αποφάσεις για κάθε πολιτική αναφοράς εξόδων και για κάθε πολιτική έγκρισης εξόδων αναφοράς που δημιουργείτε.

**Αποφάσεις:**

- Ποιο είναι το όνομα της πολιτικής;
- Σε τι χρησιμεύει η πολιτική δαπανών;
- Εάν αποφασίσατε προηγουμένως να ενεργοποιήσετε τις διεταιρικές δαπάνες, σε ποιες εταιρείες του οργανισμού σας θα ισχύει αυτή η πολιτική;
- Πότε η πολιτική είναι αποτελεσματική;
- Πότε λήγει η πολιτική;
- Ποιος είναι ο κανόνας πολιτικής;
- Ποιο είναι το αποτέλεσμα του κανόνα πολιτικής;
