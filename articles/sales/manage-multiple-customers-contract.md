# <a name="manage-multiple-customers-on-project-contracts"></a>Διαχείριση πολλών πελατών σε συμβάσεις έργου

Αυτό το θέμα παρέχει πληροφορίες σχετικά με τον τρόπο διαχείρισης πολλαπλών πελατών σε μια σύμβαση έργου. Μπορείτε να χρησιμοποιήσετε μια σύμβαση έργου όταν απαιτείται μια συμβατική συμφωνία για πολλαπλούς πελάτες για τη χρηματοδότηση μιας συμφωνίας. Στη σελίδα **Σύμβαση έργου**, η καρτέλα **Σύνοψη** περιλαμβάνει πληροφορίες σχετικά με τον κύριο πελάτη για μια συμφωνία. Είναι δυνατή η προσθήκη άλλων πελατών που συμμετέχουν στη συμφωνία στην καρτέλα **Πελάτες**.

Όλοι οι πελάτες σύμβασης στη σελίδα **Πελάτες** της προεπιλεγμένης σύμβασης έργου ως πελάτες γραμμής σύμβασης σε τυχόν νέες γραμμές σύμβασης βάσει έργου που δημιουργήθηκαν για τη σύμβαση έργου. Οποιεσδήποτε υπάρχουσες γραμμές σύμβασης βάσει έργου δεν μεταβιβάζουν καρτέλες νέων πελατών που δημιουργούνται σε μεταγενέστερο στάδιο.

Μπορείτε να προσθέσετε, να ενημερώσετε ή να διαγράψετε σύμβαση και πελάτες γραμμής σύμβασης οποιαδήποτε στιγμή προτού κερδηθεί η σύμβαση. Ένας πελάτης στη σύμβαση έργου πρέπει να οριστεί ως πελάτης στην ιδιοκτήτρια εταιρεία ή στη νομική οντότητα στη σελίδα **Πελάτες**. Οι νομικές οντότητες έχουν ρυθμιστεί στη μονάδα **Διαχείριση έργου και λογιστική** του Dynamics 365 Project Operations και είναι διαθέσιμες ως εταιρείες στις μονάδες **Πωλήσεις έργου** και **Παράδοση** του Project Operations.

## <a name="primary-customers"></a>Κύριοι πελάτες

Ο δυνητικός πελάτης στην καρτέλα **Σύνοψη** της σύμβασης έργου είναι ο κύριος πελάτης της σύμβασης. Μπορείτε να ενημερώσετε τον κύριο πελάτη από τη λίστα του πελάτη σύμβασης. Εάν προσπαθήσετε να διαγράψετε τον κύριο πελάτη από μια λίστα πελατών της σύμβασης, θα σημειωθεί σφάλμα που λέει ότι δεν είναι δυνατή η διαγραφή μιας κύριας καρτέλας πελάτη σε μια σύμβαση. Αντίθετα, αλλάξτε τον πελάτη στο πεδίο **Δυνητικός πελάτης** στην καρτέλα **Σύνοψη** της σύμβασης έργου. Όταν ενημερώνετε αυτό το πεδίο, ο πελάτης που επιλέχτηκε τελευταία προστίθεται ως νέος πελάτης σύμβασης με τη σημαία **Κύριος** ορισμένη σε **Ναι**. Ο προηγούμενος κύριος πελάτης εξακολουθεί να είναι πελάτης στη σύμβαση, ωστόσο δεν είναι πλέον ο κύριος πελάτης.

## <a name="edit-billing-split-percentages"></a>Επεξεργασία ποσοστών διαίρεσης χρέωσης

Μπορείτε να επεξεργαστείτε ποσοστά διαίρεσης χρέωσης στο πλέγμα. Όταν τα ποσοστά διαίρεσης χρέωσης δεν είναι συνολικά 100 τοις εκατό, εμφανίζεται σφάλμα. Αφού επεξεργαστείτε ένα ποσοστό διαίρεσης χρέωσης, ανανεώστε τη σελίδα **Σύμβαση έργου** για να καταργήσετε το σφάλμα.

Μπορείτε επίσης να επιλέξετε **Ομοιόμορφη διανομή** στο υποπλέγμα πελατών σύμβασης έργου. Οι διαιρέσεις χρέωσης κατανέμονται ομοιόμορφα σε όλους τους πελάτες στη σύμβαση έργου. Εάν υπάρχει συντελεστής στρογγυλοποίησης, ο συντελεστής θα προστεθεί στον πελάτη στρογγυλοποίησης. Ένας από τους πελάτες της σύμβασης έχει πάντα τη σημαία **Στρογγυλοποίηση** ορισμένη σε **Ναι**. Αυτός ο πελάτης είναι ο πελάτης στρογγυλοποίησης. Συνήθως, ο πελάτης στρογγυλοποίησης είναι επίσης ο κύριος πελάτης της σύμβασης, αλλά δεν απαιτείται.
