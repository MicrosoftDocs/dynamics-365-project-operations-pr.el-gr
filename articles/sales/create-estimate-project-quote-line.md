# <a name="estimate-a-project-quote-line"></a>Εκτίμηση μιας γραμμής προσφοράς έργου

_**Ισχύει για:** Project Operations για σενάρια βασισμένα σε πόρους/μη εφοδιασμένα_

Μια γραμμή προσφοράς βάσει έργου περιέχει λεπτομέρειες που σας βοηθούν να υπολογίσετε το κόστος και τα πιθανά έσοδα της εργασίας που σχετίζεται με την παράδοση της γραμμής προσφοράς.

Για να υπολογίσετε μια γραμμή προσφοράς βάσει έργου, στη γραμμή προσφοράς, επιλέξτε την καρτέλα **Λεπτομέρεια γραμμής προσφοράς**. Υπάρχουν δύο τρόποι για να δημιουργήσετε μια εκτίμηση για μια γραμμή προσφοράς βάσει έργου:

   - Δημιουργήστε με μη αυτόματο τρόπο την εκτίμηση απευθείας στη γραμμή προσφοράς χρησιμοποιώντας τις λεπτομέρειες της γραμμής προσφοράς. 
   - Δημιουργήστε ένα έργο και ένα σχέδιο έργου και, στη συνέχεια, συσχετίστε το έργο και τις εργασίες του έργου με τη γραμμή προσφοράς. Θα ενεργοποιηθεί η διεργασία εισαγωγής των προβλέψεων στο πλάνο έργου στη γραμμή προσφοράς με βάση τις πληροφορίες που παρείχατε.

## <a name="create-estimates-directly-on-a-project-based-quote-line"></a>Δημιουργία εκτιμήσεων απευθείας για μια γραμμή προσφοράς βάσει έργου

1. Για να δημιουργήσετε μια εκτίμηση σε μια γραμμή προσφοράς βάσει έργου, επιλέξτε την καρτέλα **Λεπτομέρεια γραμμής προσφοράς**. Το στοιχείο γραμμής που δημιουργείτε σε αυτήν την καρτέλα θα συνοψίσει την τιμή της προσφοράς για αυτήν τη γραμμή προσφοράς. 
2. Για να δημιουργήσετε λεπτομέρειες για τη γραμμή προσφοράς, επιλέξτε **Νέα λεπτομέρεια γραμμής προσφοράς** στο υποπλέγμα **Λεπτομέρειες γραμμής προσφοράς**. Θα ανοίξει ένα ρυθμιστικό γρήγορης δημιουργίας. Τα παρακάτω πεδία στη σελίδα **Γραμμή προσφοράς**.

## <a name="update-prices-on-quote-line-details"></a>Ενημέρωση τιμών στις λεπτομέρειες της γραμμής προσφοράς

Εάν έχετε αλλάξει τις τιμές στον τιμοκατάλογο έργου που επισυνάπτεται στην προσφορά ή στον τιμοκατάλογο της σύμβασης, μπορείτε να επιλέξετε **Νέος υπολογισμός** στη σελίδα **Προσφορά**, για να ανανεώσετε τις τιμές στις λεπτομέρειες της συγκεκριμένης γραμμής προσφοράς ώστε να αντικατοπτρίζεται αυτή η αλλαγή. Όταν επιλέξετε **Εκ νέου υπολογισμός**, εμφανίζεται μια προειδοποίηση που σας ενημερώνει ότι οι τιμές στις λεπτομέρειες της γραμμής προσφοράς για όλες τις γραμμές προσφοράς σε αυτήν την προσφορά θα επαναρυθμιστούν. Επιλέξτε **Ναι** για να ανανεώσετε τις τιμές τόσο για τις λεπτομέρειες της προσφοράς όσο και για τις γραμμές προσφοράς κόστους.

## <a name="access-quote-line-details-for-cost"></a>Πρόσβαση σε λεπτομέρειες της γραμμής προσφοράς για το κόστος

Για πρόσβαση στις λεπτομέρειες γραμμής προσφοράς για κόστος, ακολουθήστε τα παρακάτω βήματα:

1. Στην καρτέλα **Λεπτομέρειες γραμμής προσφοράς**, επιλέξτε μια γραμμή στο πλέγμα για να ενεργοποιήσετε ενέργειες στη γραμμή εργαλείων του υποπλέγματος. 
2. Επιλέξτε **Άνοιγμα λεπτομέρειας κόστους** για να δείτε τη σχετική χρέωση κόστους και το ποσό για αυτήν τη γραμμή προσφοράς.

> [!NOTE]
> Η αλλαγή των τιμών της μονάδας, της ποσότητας, των ημερομηνιών, του ρόλου ή της κατηγορίας στη γραμμή προσφοράς για το κόστος θα αλλάξει τις αντίστοιχες τιμές στις λεπτομέρειες της γραμμής προσφοράς για τις πωλήσεις.

## <a name="currency-on-quote-line-details-for-cost-and-sales"></a>Νόμισμα στις λεπτομέρειες της γραμμής προσφοράς για το κόστος και τις πωλήσεις

Η νομισματική μονάδα στη λεπτομέρεια γραμμής προσφοράς ορίζεται από προεπιλογή από τον τιμοκατάλογο έργου που ισχύει για την ημερομηνία έναρξης της λεπτομέρειας της γραμμής προσφοράς.

Η νομισματική μονάδα στη λεπτομέρεια γραμμής προσφοράς για κόστος ορίζεται από προεπιλογή από τον τιμοκατάλογο της συμβαλλόμενης μονάδας της προσφοράς που ισχύει από την ημερομηνία έναρξης της λεπτομέρειας της γραμμής προσφοράς για κόστος.

> [!NOTE]
> Οι υπολογισμοί κερδοφορίας μετατρέπουν το ποσό στις λεπτομέρειες της γραμμής προσφοράς για το κόστος και τις πωλήσεις στη βασική νομισματική μονάδα του περιβάλλοντος για την αναφορά του συνολικού εκτιμώμενου περιθωρίου στην προσφορά. Τα σφάλματα στρογγυλοποίησης νομισμάτων και τα τροποποιημένα περιθώρια θα μπορούσαν να προκύψουν λόγω έλλειψης ημερομηνιών ως αποτελεσματικών συναλλαγματικών ισοτιμιών. Χρησιμοποιήστε αυτούς τους υπολογισμούς μόνο σε προσφορές έργου, καθώς πρόκειται για προσεγγίσεις και δεν είναι πραγματικές αναφορές ή άλλες αναφορές που απαιτούν μεγαλύτερη ακρίβεια στρογγυλοποίησης και επίγνωση της ισχύος της ημερομηνίας για τις συναλλαγματικές ισοτιμίες.
