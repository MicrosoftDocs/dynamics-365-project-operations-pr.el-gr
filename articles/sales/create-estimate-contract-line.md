# <a name="estimate-a-project-contract-line"></a>Εκτίμηση μιας γραμμής σύμβασης έργου

_**Ισχύει για:** Project Operations για σενάρια βασισμένα σε πόρους/μη εφοδιασμένα_ 

Στο Dynamics 365 Project Operations, μια γραμμή σύμβασης βάσει έργου έχει λεπτομέρειες που σας βοηθούν να υπολογίσετε το κόστος και τα πιθανά έσοδα των σχετικών εργασιών για την παροχή της γραμμής σύμβασης.

Για να υπολογίσετε μια γραμμή σύμβασης βάσει έργου, μεταβείτε στην καρτέλα **Λεπτομέρεια γραμμής σύμβασης** στη γραμμή σύμβασης βάσει έργου **Γραμμή σύμβασης**.  Υπάρχουν δύο τρόποι για τη δημιουργία μιας εκτίμησης σε μια γραμμή σύμβασης βάσει έργου:

   - Δημιουργήστε μια εκτίμηση απευθείας στη γραμμή σύμβασης προσθέτοντας με μη αυτόματο τρόπο τις λεπτομέρειες της γραμμής σύμβασης.
   - Δημιουργήστε ένα έργο και ένα σχέδιο έργου και, στη συνέχεια, συσχετίστε το έργο και τις εργασίες στη γραμμή σύμβασης του έργου. Αυτό ενεργοποιεί τη διεργασία με την οποία μπορείτε να εισαγάγετε την εκτίμηση σχεδίου έργου στη γραμμή σύμβασης με βάση τα στοιχεία που περιλαμβάνονται στη γραμμή σύμβασης.

## <a name="create-an-estimate-directly-on-a-project-contract-line"></a>Δημιουργία μιας εκτίμησης απευθείας σε μια γραμμή σύμβασης έργου

Για δημιουργία μιας εκτίμησης απευθείας σε μια γραμμή σύμβασης έργου, ακολουθήστε αυτά τα βήματα:

1. Μεταβείτε στη γραμμή σύμβασης και επιλέξτε την καρτέλα **Λεπτομέρεια γραμμής σύμβασης**. Οι γραμμές που δημιουργείτε σε αυτήν την καρτέλα συνοψίζονται και εμφανίζονται ως η **Συμβατική τιμή** για αυτήν τη **Γραμμή σύμβασης**. 
2. Στο υποπλέγμα **Λεπτομέρειες γραμμής σύμβασης**, επιλέξτε **Νέα λεπτομέρεια γραμμής σύμβασης**. Ανοίγει ένα ρυθμιστικό γρήγορης δημιουργίας. Τα παρακάτω πεδία είναι διαθέσιμα στη σελίδα **Λεπτομέρειες γραμμής σύμβασης**.

## <a name="update-prices-on-contract-line-details"></a>Ενημέρωση τιμών στις λεπτομέρειες της γραμμής σύμβασης

Εάν αλλάξετε τις τιμές στον τιμοκατάλογο έργου που επισυνάπτεται στη σύμβαση ή τον τιμοκατάλογο της σύμβασης, μπορείτε να ανανεώσετε τις τιμές στις μεμονωμένες λεπτομέρειες της σύμβασης για να απεικονίσετε την αλλαγή. Στη σελίδα **Σύμβαση**, επιλέξτε **Εκ νέου υπολογισμός**. Εμφανίζεται μια προειδοποίηση που σας ενημερώνει ότι θα γίνει επαναφορά των τιμών για όλες τις γραμμές σύμβασης αυτής της σύμβασης. Επιλέξτε **Ναι** για να ανανεώσετε τις τιμές τόσο για τις λεπτομέρειες της σύμβασης πωλήσεων όσο και για τις γραμμές σύμβασης κόστους.

## <a name="access-contract-line-details-for-cost"></a>Πρόσβαση σε λεπτομέρειες της γραμμής σύμβασης για το κόστος

Στην καρτέλα **Λεπτομέρειες γραμμής σύμβασης**, επιλέξτε μια γραμμή στο πλέγμα για να εμφανίσετε ενέργειες στη γραμμή εργαλείων του υποπλέγματος. Η πρώτη ενέργεια στη γραμμή εργαλείων υποπλεγματος είναι **Άνοιγμα λεπτομέρειας κόστους**. Για να δείτε τη σχετική χρέωση κόστους και το ποσό για αυτήν τη λεπτομέρεια της γραμμής σύμβασης, επιλέξτε **Άνοιγμα λεπτομερειών κόστους**. 

> [!NOTE]
> Η αλλαγή της εταιρείας πόρων, της μονάδας πόρων, της ποσότητας, των ημερομηνιών, του ρόλου ή των τιμών κατηγοριών στη λεπτομέρεια της γραμμής σύμβασης για το **Κόστος** αλλάζει επίσης τις αντίστοιχες τιμές στη λεπτομέρεια της γραμμής σύμβασης για **Πωλήσεις**.

## <a name="currency-on-contract-line-details-for-cost-and-sales"></a>Λεπτομέρειες νομίσματος στις γραμμές σύμβασης για το κόστος και τις πωλήσεις

Η λεπτομέρεια της γραμμής σύμβασης για τις **Πωλήσεις** ορίζει την προεπιλεγμένη νομισματική μονάδα από τον τιμοκατάλογο έργου που ισχύει για την ημερομηνία έναρξης της λεπτομέρειας γραμμής σύμβασης.

Η λεπτομέρεια της γραμμής σύμβασης για το **Κόστος** ορίζει την προεπιλεγμένη νομισματική μονάδα από τον τιμοκατάλογο της αναθέτουσας μονάδας της σύμβασης που ισχύει για την ημερομηνία έναρξης της λεπτομέρειας γραμμής σύμβασης για το **Κόστος**.

Οι υπολογισμοί κερδοφορίας μετατρέπουν τα ποσά για τις λεπτομέρειες της γραμμής σύμβασης για το **Κόστος** και τις **Πωλήσεις** στη βασική νομισματική μονάδα του περιβάλλοντος για την αναφορά των συνολικών πραγματικών και εκτιμώμενων περιθωρίων στη σύμβαση.

> [!NOTE]
> Τα σφάλματα στρογγυλοποίησης νομισμάτων και τα τροποποιημένα περιθώρια θα μπορούσαν να προκύψουν λόγω έλλειψης ημερομηνιών ως αποτελεσματικών συναλλαγματικών ισοτιμιών. Χρησιμοποιήστε αυτούς τους υπολογισμούς μόνο σε συμβάσεις έργου, καθώς πρόκειται για προσεγγίσεις και δεν είναι για πραγματικές αναφορές ή άλλες αναφορές που απαιτούν μεγαλύτερη ακρίβεια στρογγυλοποίησης και επίγνωση της ισχύος της ημερομηνίας για τις συναλλαγματικές ισοτιμίες.
