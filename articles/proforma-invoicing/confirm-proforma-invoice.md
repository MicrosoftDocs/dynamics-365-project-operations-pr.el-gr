# <a name="confirm-a-proforma-invoice"></a>Επιβεβαίωση προτιμολογίου

_**Ισχύει για:** Project Operations για σενάρια βασισμένα σε πόρους/μη εφοδιασμένα_

Αφού επιβεβαιωθεί ένα προτιμολόγιο, η κατάσταση των ενημερώσεων του τιμολογίου έργου θα είναι **Επιβεβαιώθηκε**. Όταν επιβεβαιώνεται ένα τιμολόγιο, γίνεται μόνο για ανάγνωση. Προχωρώντας, το τιμολόγιο μπορεί να διορθωθεί μόνο εάν υπάρχουν διορθώσεις ή πιστώσεις που έχουν προετοιμαστεί από τον πελάτη ή όταν έχει επισημανθεί ως πληρωμένο.

Στον ακόλουθο πίνακα παρουσιάζονται οι πραγματικές τιμές που δημιουργούνται από το σύστημα. Αυτές οι πραγματικές τιμές δημιουργούνται όταν εκτελούνται συγκεκριμένες λειτουργίες στο προσχέδιο τιμολογίου πριν επιβεβαιωθεί.
