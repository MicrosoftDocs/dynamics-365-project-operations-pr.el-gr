# <a name="provision-a-new-environment"></a>Παροχή ενός νέου περιβάλλοντος

_**Ισχύει για:** Project Operations για σενάρια βασισμένα σε πόρους/μη εφοδιασμένα_

Αυτό το θέμα παρέχει πληροφορίες σχετικά με τον τρόπο παροχής ενός νέου περιβάλλοντος Dynamics 365 Project Operations για σενάρια βασισμένα σε πόρους/μη εφοδιασμένα.

## <a name="enable-project-operations-automated-provisioning-in-an-lcs-project"></a>Ενεργοποίηση αυτόματης παροχής Project Operations σε ένα έργο LCS

Χρησιμοποιήστε τα παρακάτω βήματα για να ενεργοποιήσετε την αυτοματοποιημένη ροή παροχής Project Operations για το έργο LCS.

1. Μεταβείτε στο [LCS](https://lcs.dynamics.com/v2) και επιλέξτε το πλακίδιο **Διαχείριση δυνατοτήτων προεπισκόπησης**.
2. Στη λίστα **Δυνατότητα προεπισκόπησης**, επιλέξτε **Project Operations** και **Η δυνατότητα προεπισκόπησης έχει ενεργοποιηθεί** για την ενεργοποίηση του Project Operations.

> [!NOTE]
> Αυτό το βήμα εκτελείται μόνο μία φορά ανά έργο LCS.

## <a name="provision-a-project-operations-environment"></a>Παροχή περιβάλλοντος Project Operations

1. Ανοίξτε ένα νέο Dynamics 365 Finance [περιβάλλον επίδειξης](https://docs.microsoft.com/dynamics365/fin-ops-core/dev-itpro/deployment/deploy-demo-environment) ή μια ανάπτυξη [περιβάλλοντος προστατευμένης εκτέλεσης / παραγωγής](https://docs.microsoft.com/edynamics365/fin-ops-core/dev-itpro/deployment/deployenvironment-newinfrastructure). 
2. Περιηγηθείτε στον οδηγό **Παροχής περιβάλλοντος**. 

> [!IMPORTANT]
> Βεβαιωθείτε ότι η επιλεγμένη έκδοση της εφαρμογής είναι 10.0.13 ή μεταγενέστερη.

3. Για την παροχή του Project Operations, στην περιοχή **Ρυθμίσεις για προχωρημένους**, επιλέξτε **Common Data Service**. 
4. Ενεργοποιήστε τη **Ρύθμιση Common Data Service** επιλέγοντας **Ναι** και, στη συνέχεια, καταχωρίστε πληροφορίες στα απαιτούμενα πεδία:

  - Ονομασία
  - Περιοχή
  - Γλώσσα
  - Νομισματική μονάδα
 
5. Στο πεδίο **Πρότυπο Common Data Service**, επιλέξτε **Project Operations** 

6. Επιλέξτε τον τύπο περιβάλλοντος για την ανάπτυξή σας. Μια δοκιμαστική έκδοση με συνδρομή θα σας επιτρέψει να αναπτύξετε ένα περιβάλλον CDS για 30 ημέρες. 

![Ρυθμίσεις ανάπτυξης](./media/1DeploymentSettings.png)

> [!IMPORTANT]
> Επιλέξτε **Συμφωνώ** για αποδοχή των όρων της υπηρεσίας και, στη συνέχεια, επιλέξτε **Τέλος** για να επιστρέψετε στις ρυθμίσεις ανάπτυξης.

![Συναίνεση ανάπτυξης](./media/2DeploymentConsent.png)

7. Συμπληρώστε τα υπόλοιπα απαιτούμενα πεδία στον οδηγό και επιβεβαιώστε την ανάπτυξη. Ο χρόνος παροχής περιβάλλοντος ποικίλλει ανάλογα με τον τύπο περιβάλλοντος. Η παροχή μπορεί να διαρκέσει έως και έξι ώρες.

  Αφού ολοκληρωθεί με επιτυχία η ανάπτυξη, το περιβάλλον θα εμφανίζεται ως **Αναπτυγμένο**.

8. Για να επιβεβαιώσετε ότι το περιβάλλον έχει αναπτυχθεί με επιτυχία, επιλέξτε το στοιχείο **Σύνδεση** και συνδεθείτε στο περιβάλλον για επιβεβαίωση.

![Λεπτομέρειες περιβάλλοντος ](./media/3EnvironmentDetails.png)

## <a name="apply-project-operations-finance-demo-data-optional-step"></a>Εφαρμογή των δεδομένων επίδειξης Project Operations Finance (προαιρετικό βήμα)

Εφαρμόστε τα δεδομένα επίδειξης του Project Operations Finance σε έκδοση υπηρεσίας 10.0.13 του φιλοξενούμενου περιβάλλοντος όπως περιγράφεται σε [αυτό το άρθρο](resource-apply-finance-demo-data.md).

## <a name="apply-updates-to-the-finance-environment"></a>Εφαρμογή ενημερώσεων στο περιβάλλον Finance

Για το Project Operations απαιτείται ένα Finance περιβάλλον με έκδοση εφαρμογής **10.0.13 (10.0.569.20009)** ή νεότερη έκδοση.

Μπορεί να χρειαστεί να εφαρμόσετε ενημερωμένες εκδόσεις ποιότητας στο περιβάλλον Finance για να λάβετε αυτήν την έκδοση.

1. Στο LCS, στη σελίδα **Λεπτομέρειες περιβάλλοντος**, στην ενότητα **Διαθέσιμες ενημερώσεις**, επιλέξτε **Προβολή ενημέρωσης**.

![Προβολή ενημερώσεων](./media/5ViewUpdates.png)

2. Στη σελίδα **Δυαδικές ενημερώσεις**, επιλέξτε **Αποθήκευση πακέτου**.

![Αποθήκευση πακέτου](./media/6SavePackage.png)

3. Επιλέξτε **Επιλογή όλων** και μετά **Αποθήκευση πακέτου**.

![Έλεγχος και αποθήκευση ενημερώσεων](./media/7ReviewAndSaveUpdates.png)

4. Καταχωρίστε ένα όνομα και μια περιγραφή του πακέτου και, στη συνέχεια, επιλέξτε **Αποθήκευση**. Ανάλογα με τη σύνδεση στο Internet, αυτή η διαδικασία μπορεί να διαρκέσει αρκετό χρόνο.

![Αποστολή πακέτου στη βιβλιοθήκη παγίων στοιχείων](./media/8UploadPackageToAssetsLibrary.png)

5. Μετά την αποθήκευση του πακέτου, επιλέξτε **Τέλος** και αποθηκεύστε αυτό το πακέτο στη βιβλιοθήκη παγίων στοιχείων του έργου σας στο LCS.

Η αποθήκευση και η επικύρωση του πακέτου μπορεί να διαρκέσει ~ 15 λεπτά.

6. Για να εφαρμόσετε την ενημέρωση, μεταβείτε στη σελίδα **Λεπτομέρειες περιβάλλοντος** στο LCS και επιλέξτε **Συντήρηση** > **Εφαρμογή ενημερώσεων**.

![Διατήρηση περιβαλλόντων](./media/9MaintainEnvironment.png)

7. Στη λίστα ενημερωμένων εκδόσεων, επιλέξτε το πακέτο που δημιουργήσατε και επιλέξτε **Εφαρμογή**.

![Εφαρμογή ενημερώσεων](./media/10ApplyUpdates.png)

Η εξυπηρέτηση περιβάλλοντος θα διαρκέσει αρκετό χρόνο. Αφού ολοκληρωθεί, το περιβάλλον θα επιστρέψει σε μια αναπτυγμένη κατάσταση.

![Ανεπτυγμένο περιβάλλον](./media/11EnvironmentDeployed.png)

## <a name="establish-a-dual-write-connection"></a>Δημιουργία σύνδεσης διπλής εγγραφής 

1. Στο έργο σας LCS, μεταβείτε στη σελίδα **Λεπτομέρειες περιβάλλοντος**.
2. Στην περιοχή **Πληροφορίες περιβάλλοντος Common Data Service**, επιλέξτε **Σύνδεση σε CD για εφαρμογές**.
3. Αφού ολοκληρωθεί η σύνδεση, επιλέξτε ξανά **Σύνδεση σε CD για εφαρμογές**. Θα ανακατευθυνθείτε για διπλή εγγραφή στο Finance.

![Σύνδεση προς το CDS](./media/12LinktoCDS.png)

4. Επιλέξτε **Εφαρμογή λύσης** για να αποκτήσετε πρόσβαση στις οντότητες που θα αντιστοιχιστούν στην ενοποίηση.

![Εφαρμογή λύσεων](./media/13ApplySolutions.png)

5. Επιλέξτε και τις δύο λύσεις, **Αντιστοίχιση οντότητας διπλής εγγραφής Dynamics 365 Finance and Operations** και **Αντιστοιχίσεις οντότητας διπλής εγγραφής Dynamics 365 Project Operations** και, στη συνέχεια, επιλέξτε **Εφαρμογή**.

![Επιβεβαίωση λύσεων](./media/14ConfirmSolutions.png)

Αφού εφαρμοστούν οι λύσεις, οι οντότητες διπλής εγγραφής εφαρμόζονται στο περιβάλλον.

![Εφαρμογή λύσεων](./media/15ApplyingSolutions.png)

Αφού εφαρμοστούν οι οντότητες, όλες οι διαθέσιμες αντιστοιχίσεις παρατίθενται στο περιβάλλον.

![Αντιστοιχίσεις διπλής εγγραφής](./media/15DWMappings.png)

## <a name="refresh-the-data-entities-after-the-update"></a>Ανανέωση των οντοτήτων δεδομένων μετά την ενημέρωση

1. Στο Finance, μεταβείτε στον χώρο εργασίας **Διαχείριση δεδομένων**.

![Χώρος εργασίας διαχείρισης δεδομένων](./media/16DataManagement.png)

2. Επιλέξτε το πλακίδιο **Παράμετροι πλαισίου**.

![Παράμετροι πλαισίου](./media/17FrameworkParameters.png)

3. Στη σελίδα **Ρυθμίσεις οντότητας**, επιλέξτε **Ανανέωση λίστας οντοτήτων**.

![Ανανέωση λίστας οντοτήτων](./media/18RefreshEntityList.png)

Η ανανέωση θα διαρκέσει περίπου 20 λεπτά. Όταν ολοκληρωθεί, θα λάβετε μια ειδοποίηση.

![Επιβεβαίωση ανανέωσης](./media/19RefreshConfirmation.png)

## <a name="run-project-operations-dual-write-maps"></a>Εκτέλεση αντιστοιχίσεων διπλής εγγραφής Project Operations

1. Στο έργο σας LCS, μεταβείτε στη σελίδα **Λεπτομέρειες περιβάλλοντος**.
2. Στην περιοχή **Πληροφορίες περιβάλλοντος Common Data Service**, επιλέξτε **Σύνδεση σε CD για εφαρμογές**. Αφού επιλέξετε τη σύνδεση, θα ανακατευθυνθείτε στη λίστα με τις οντότητες στις αντιστοιχίσεις.
3. Ξεκινήστε τις αντιστοιχίσεις όπως περιγράφεται στον ακόλουθο πίνακα. Βεβαιωθείτε ότι έχετε ακολουθήσει την ακολουθία όπως εμφανίζεται στη λίστα.

| **Αντιστοίχιση οντότητας** | **Ανανέωση οντότητας** | **Αρχικός συγχρονισμός** | **Κύρια για αρχικό συγχρονισμό** | **Προϋποθέσεις εκτέλεσης** | **Προϋποθέσεις για αρχικό συγχρονισμό** |
| --- | --- | --- | --- | --- | --- |
| **Ρόλοι πόρων έργου για όλες τις εταιρείες (bookableresourcecategories)** | No | Ναι | Common Data Service | No | Μη διαθέσιμο |
| **Νομικές οντότητες (cdm\_εταιρείες)** | No | Ναι | Εφαρμογές Finance and Operations | No | Μη διαθέσιμο |
| **Πραγματικές τιμές ενοποίησης Project Operations (msdyn\_πραγματικές τιμές)** | No | No | Μη διαθέσιμο | Ναι | No |
| **Γραμμές σύμβασης βάσει έργου (salesorderdetails)** | No | No | Μη διαθέσιμο | No | No |
| **Οντότητα ενοποίησης για σχέσεις συναλλαγών έργου (msdyn\_transactionconnections)** | No | No | Μη διαθέσιμο | No | Μη διαθέσιμο |
| **Ορόσημα γραμμής σύμβασης ενοποίησης Project Operations (msdyn\_contractlinesscheduleofvalues)** | No | No | Μη διαθέσιμο | No | Μη διαθέσιμο |
| **Οντότητα ενοποίησης Project Operations για εκτιμήσεις δαπανών (msdyn\_estimateslines)** | No | No | Μη διαθέσιμο | No | Μη διαθέσιμο |
| **Οντότητα ενοποίησης Project Operations για ωριαίες εκτιμήσεις (msdyn\_resourceassignments)** | No | No | Μη διαθέσιμο | No | Μη διαθέσιμο |
| **Οντότητα εξαγωγής δαπανών έργου ενοποίησης Project Operations (msdyn\_έξοδα)** | Ναι | No | Μη διαθέσιμο | No | Μη διαθέσιμο |
| **Οντότητα ενοποίησης Project Operations για ωριαίες εκτιμήσεις (msdyn\_resourceassignments)** | Ναι | No | Μη διαθέσιμο | No | Μη διαθέσιμο |

4. Για να ανανεώσετε την οντότητα, επιλέξτε το όνομα της αντιστοίχισης και, στη συνέχεια, επιλέξτε **Ανανέωση οντοτήτων**. 
5. Προχωρήστε με την εκτέλεση της αντιστοίχισης μετά την ολοκλήρωση της ανανέωσης.

![Ανανέωση αντιστοίχισης](./media/20RefreshMapping.png)

Πριν ενεργοποιήσετε την επόμενη αντιστοίχιση, επαληθεύστε ότι η αντιστοίχιση στον πίνακα βρίσκεται σε κατάσταση **Εκτελείται**. Η εκτέλεση αντιστοιχίσεων με μεγαλύτερο αριθμό προϋποθέσεων ενδέχεται να διαρκέσει αρκετό χρόνο.

Για να εκτελεστεί μια αντιστοίχιση με προϋποθέσεις, ενεργοποιήστε την εναλλαγή **Εμφάνιση σχετικών αντιστοιχίσεων οντοτήτων**. Εάν ο πίνακας υποδεικνύει ότι οι **Προϋποθέσεις για αρχικό συγχρονισμό** είναι **Όχι**, βεβαιωθείτε ότι η σημαία **Αρχικός συγχρονισμός** είναι **Απενεργοποιημένη** σε όλες τις προαπαιτούμενες αντιστοιχίσεις πριν να τον πραγματοποιήσετε.

![Εκτέλεση αντιστοίχισης](./media/21RunMap.png)

6. Επικυρώστε όλες τις αντιστοιχίσεις που σχετίζονται με το έργο βρίσκονται σε κατάσταση "εκτέλεση".

![Όλοι οι αντιστοιχίσεις που εκτελούνται](./media/22AllMapsRunning.png)

Το περιβάλλον Project Operations έχει πλέον δοθεί και ρυθμιστεί.
