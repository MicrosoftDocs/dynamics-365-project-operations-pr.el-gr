# <a name="vendor-invoice-lines-for-time"></a>Γραμμές τιμολογίου πωλητή για χρόνο

_**Ισχύει για:** Ελαφριά ανάπτυξη - συμφωνία για προτιμολόγηση_

Ένα τιμολόγιο πωλητή στο Microsoft Dynamics 365 Project Operations μπορεί να έχει γραμμές τιμολογίου πωλητή για χρόνο. Οι διαχειριστές έργων μπορούν να χρησιμοποιήσουν γραμμές τιμολογίου πωλητή για χρόνο, για να καταγράψουν το κόστος χρόνου υπεργολάβου σε έργα.

Οι γραμμές τιμολογίου του προμηθευτή για χρόνο ενδέχεται να αναφέρονται ή όχι σε μια γραμμή υπεργολαβίας για χρόνο. Εάν μια γραμμή τιμολογίου πωλητή για χρόνο αναφέρεται σε υπεργολαβία, οι διαχειριστές έργου θα μπορούν να αντιστοιχιστούν και να επαληθεύσουν τον χρόνο που τιμολογούνται από τη γραμμή τιμολογίου του πωλητή με βάση τον χρόνο που καταγράφεται από υπεργολάβους και εγκρίνεται από διαχειριστές έργου στο έργο.

Ο ακόλουθος πίνακας παρέχει πληροφορίες σχετικά με τα πεδία στις γραμμές τιμολογίου προμηθευτή για χρόνο.

| Πεδίο | Description | Αντίκτυπος στη λειτουργία |
| --- | --- | --- |
| 'Ονομα | Το όνομα της γραμμής τιμολογίου πωλητή που θα βοηθήσει με την αναγνώριση. | Αυτό το όνομα θα εμφανίζεται ως η πρώτη στήλη σε όλες τις αναζητήσεις που βασίζονται σε γραμμές τιμολογίου πωλητή. |
| Description | Μια σύντομη περιγραφή των υπηρεσιών που τιμολογούνται από τον πωλητή στη γραμμή τιμολογίου του πωλητή. | Κανένα |
| Υπεργολαβία | Η υπεργολαβία στην οποία παραγγέλθηκαν αρχικά οι υπηρεσίες. | Όταν επιλέγεται μια υπεργολαβία για το τιμολόγιο πωλητή, σε όλες τις γραμμές του τιμολογίου πωλητή θα μεταβιβάζεται αυτή η επιλογή. Ένα τιμολόγιο πωλητή δεν μπορεί να έχει γραμμές τιμολογίου πωλητή που να αναφέρονται σε διαφορετικές υπεργολαβίες. |
| Γραμμή υπεργολαβίας | Η γραμμή υπεργολαβίας στην οποία παραγγέλθηκαν οι υπηρεσίες. Η λίστα των γραμμών υπεργολαβίας που μπορούν να επιλεγούν περιορίζεται στις γραμμές στην επιλεγμένη υπεργολαβία. | Όταν επιλέγεται μια γραμμή υπεργολαβίας σε μια γραμμή τιμολογίου πωλητή για χρόνο, εισάγονται από τα αντίστοιχα πεδία της γραμμής υπεργολαβίας προεπιλεγμένες τιμές για τα πεδία **Έργο**, **Ρόλος** και **Πόρος με δυνατότητα κράτησης**. Εάν η επιλεγμένη γραμμή υπεργολαβίας έχει τιμές στα πεδία **Έργο**, **Ρόλος** και **Με δυνατότητα κράτησης**, οι τιμές των αντίστοιχων πεδίων στη γραμμή τιμολογίου του πωλητή δεν μπορούν να διαφέρουν από αυτές τις τιμές. |
| Ημερομηνία συναλλαγής | Η ημερομηνία κατά την οποία το πραγματικό κόστος της γραμμής τιμολογίου πωλητή θα καταγραφεί στο έργο. | Κανένα |
| Κλάση συναλλαγής | Η προεπιλεγμένη τιμή είναι **Χρόνος**. | Η τιμή **Χρόνος** δηλώνει ότι η γραμμή τιμολογίου του προμηθευτή χρησιμοποιείται για την καταγραφή του ποσού τιμολογίου χρόνου υπεργολάβου. |
| Project | Το όνομα του έργου στο οποίο χρησιμοποιήθηκαν οι υπηρεσίες που τιμολογήθηκαν. | Αυτό το πεδίο είναι υποχρεωτικό και δεν είναι δυνατό να μείνει κενό. |
| Κλείσιμο εργασίας | Το όνομα της εργασίας έργου στο οποίο χρησιμοποιήθηκαν οι υπηρεσίες που τιμολογήθηκαν. Αυτό το πεδίο είναι διαθέσιμο μόνο εάν έχει επιλεγεί ένα έργο. Η επιλογή μιας εργασίας έργου είναι προαιρετική. | Εάν αυτό το πεδίο μείνει κενό, ο διαχειριστής έργου μπορεί να αντιστοιχεί στη γραμμή τιμολογίου πωλητή σε χρόνο που καταγράφεται από πόρους υπεργολάβου σε οποιαδήποτε εργασία του έργου. Εάν η γραμμή τιμολογίου του πωλητή δεν αναφέρεται σε μια γραμμή υπεργολαβίας και αυτό το πεδίο μένει κενό, το πραγματικό κόστος που δημιουργείται από τη γραμμή τιμολογίου του πωλητή δεν θα συνδεθεί με οποιεσδήποτε μη μειωμένες πραγματικές τιμές πωλήσεων. Σε αυτή την περίπτωση, εάν έχει ρυθμιστεί χρέωση βάσει εργασιών, το κόστος ενδεχομένως να μην είναι δυνατό να τιμολογηθεί στον τελικό πελάτη. |
| Ρόλος | Ο ρόλος των πόρων υπεργολαβίας των οποίων ο χρόνος τιμολογείται. | Το πεδίο αυτό καθορίζει τον ρόλο που εκτελείται από τους πόρους υπεργολαβίας, του οποίου ο χρόνος τιμολογείται στο τιμολόγιο του πωλητή. |
| Πόρος με δυνατότητα κράτησης | Το όνομα του υπεργολάβου ο χρόνος του οποίου τιμολογείται. Η επιλογή ενός πόρου με δυνατότητα κράτησης είναι προαιρετική. | Εάν αυτό το πεδίο μείνει κενό, ο διαχειριστής έργου μπορεί να αντιστοιχεί στη γραμμή τιμολογίου πωλητή σε χρόνο που καταγράφεται από οποιονδήποτε πόρο ανήκει στον προμηθευτή στη γραμμή τριμολογίου προμηθευτή. |
| Ποσότητα | Εισαγάγετε τον αριθμό ωρών του χρόνου που τιμολογούνται από τον πωλητή στη γραμμή τιμολογίου. |Κανένα |
| Ομάδα μονάδων | Η προεπιλεγμένη τιμή είναι **ομάδα μονάδας χρόνου** και δεν μπορεί να αλλάξει. | Κανένα |
| Μονάδα | Η προεπιλεγμένη τιμή είναι η βασική μονάδα ωρών από την ομάδα μονάδας χρόνου. Μπορείτε να αλλάξετε αυτήν την τιμή για να αγοράσετε οποιαδήποτε μονάδα της ομάδας μονάδας χρόνου, για παράδειγμα, ημέρα ή εβδομάδα. | Ο συνδυασμός των τιμών **Ρόλος** και **Μονάδα** θα χρησιμοποιηθεί ως προεπιλεγμένη ή υπολογισμένη τιμή για το πεδίο **Τιμή μονάδας** στη γραμμή τιμολογίου προμηθευτή. |
| Τιμή μονάδας | Η προεπιλεγμένη τιμή μονάδας χρησιμοποιεί τον συνδυασμό των τιμών **Ρόλος** και **μονάδα** από τον τιμοκατάλογο έργου που ισχύει για την ημερομηνία συναλλαγής της γραμμής τιμολογίου πωλητή. | Εάν η τιμή για τον ισχύοντα τιμοκατάλογο έργου έχει οριστεί σε μια μονάδα που διαφέρει από τη μονάδα στη γραμμή τιμολογίου του πωλητή, το σύστημα χρησιμοποιεί τη μετατροπή μονάδας για τον υπολογισμό της τιμής ανά μονάδα. |
| Υποσύνολο | Αυτό το πεδίο μόνο για ανάγνωση υπολογίζεται ως *Ποσότητα* &times; *Τιμή μονάδας*, εάν οι τιμές εισάγονται και στο πεδίο **Ποσότητα** και στο πεδίο **Τιμή μονάδας**. Εάν ένα ή και τα δύο αυτά πεδία είναι κενά, μπορείτε να καταχωρήσετε μια τιμή σε αυτό το πεδίο. | Κανένα |
| Φόρος πωλήσεων | Καταχωρήστε το ποσό του φόρου πωλήσεων. | Κανένα |
| Συνολικό ποσό | Το συνολικό ποσό της γραμμής τιμολογίου του πωλητή, συμπεριλαμβανομένων των φόρων. Αυτό το πεδίο υπολογίζεται ως *Υποσύνολο* + *Φόρος πωλήσεων*. | Κανένα |
