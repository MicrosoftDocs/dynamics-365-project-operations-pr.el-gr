# <a name="vendor-invoice-lines-for-products"></a>Γραμμές τιμολογίου πωλητή για προϊόντα

_**Ισχύει για:** Ελαφριά ανάπτυξη - συμφωνία για προτιμολόγηση_

Ένα τιμολόγιο πωλητή στο Microsoft Dynamics 365 Project Operations μπορεί να έχει γραμμές τιμολογίου πωλητή για προϊόντα (αναφέρεται επίσης ως υλικό). Οι διαχειριστές έργων μπορούν να χρησιμοποιήσουν γραμμές τιμολογίου πωλητή για τα προϊόντα, για να καταγράψουν το κόστος των προϊόντων που αγοράστηκαν σε έργα.

Οι γραμμές τιμολογίου του προμηθευτή για προϊόντα ενδέχεται να αναφέρονται ή όχι σε μια γραμμή υπεργολαβίας για υλικά. Εάν μια γραμμή τιμολογίου πωλητή για προϊόντα αναφέρεται σε υπεργολαβία, οι διαχειριστές έργου θα μπορούν να αντιστοιχιστούν και να επαληθεύσουν τα προϊόντα που τιμολογούνται από τη γραμμή τιμολογίου του πωλητή με βάση τη χρήση τν αγορασμένων προϊόντων που καταγράφονται και έχουν εγκριθεί από τους διαχειριστές έργου.

Ο ακόλουθος πίνακας παρέχει πληροφορίες σχετικά με τα πεδία στις γραμμές τιμολογίου προμηθευτή για προϊόντα.

| Πεδίο | Description | Αντίκτυπος στη λειτουργία |
| --- | --- | --- |
| 'Ονομα | Το όνομα της γραμμής τιμολογίου πωλητή που θα βοηθήσει με την αναγνώριση. | Αυτό το όνομα θα εμφανίζεται ως η πρώτη στήλη σε όλες τις αναζητήσεις που βασίζονται σε γραμμές τιμολογίου πωλητή. |
| Description | Μια σύντομη περιγραφή των προϊόντων που τιμολογούνται από τον πωλητή στη γραμμή τιμολογίου του πωλητή. | Κανένα |
| Υπεργολαβία | Η υπεργολαβία στην οποία παραγγέλθηκαν αρχικά τα προϊόντα. | Όταν επιλέγεται μια υπεργολαβία για το τιμολόγιο πωλητή, σε όλες τις γραμμές του τιμολογίου πωλητή θα μεταβιβάζεται αυτή η επιλογή. Ένα τιμολόγιο πωλητή δεν μπορεί να έχει γραμμές τιμολογίου πωλητή που να αναφέρονται σε διαφορετικές υπεργολαβίες. |
| Γραμμή υπεργολαβίας | Η γραμμή υπεργολαβίας στην οποία παραγγέλθηκαν τα προϊόντα. Η λίστα των γραμμών υπεργολαβίας που μπορούν να επιλεγούν περιορίζεται στις γραμμές στην επιλεγμένη υπεργολαβία. | Όταν επιλέγεται μια γραμμή υπεργολαβίας σε μια γραμμή τιμολογίου πωλητή για προϊόντα, εισάγονται από τα αντίστοιχα πεδία της γραμμής υπεργολαβίας προεπιλεγμένες τιμές για τα πεδία **Έργο**, **Εργασία** και **Προϊόντα**. Εάν η επιλεγμένη γραμμή υπεργολαβίας έχει τιμές στα πεδία **Έργο**, **Εργασία** και **Προϊόν**, οι τιμές των αντίστοιχων πεδίων στη γραμμή τιμολογίου του πωλητή δεν μπορούν να διαφέρουν από αυτές τις τιμές. |
| Ημερομηνία συναλλαγής | Η ημερομηνία κατά την οποία το πραγματικό κόστος της γραμμής τιμολογίου πωλητή θα καταγραφεί στο έργο. | Κανένα|
| Κλάση συναλλαγής | Όταν τιμολογώνται τα προϊόντα, αυτό το πεδίο πρέπει να οριστεί σε **Υλικό**. | Η τιμή **Υλικό** δηλώνει ότι η γραμμή τιμολογίου του προμηθευτή χρησιμοποιείται για την καταγραφή του ποσού τιμολογίου για υλικό που αγοράστηκε. |
| Project | Το όνομα του έργου στο οποίο χρησιμοποιήθηκαν τα προϊόντα που τιμολογήθηκαν. | Αυτό το πεδίο είναι υποχρεωτικό και δεν είναι δυνατό να μείνει κενό. |
| Κλείσιμο εργασίας | Το όνομα της εργασίας του έργου στο οποίο χρησιμοποιήθηκαν τα προϊόντα που τιμολογήθηκαν. Αυτό το πεδίο είναι διαθέσιμο μόνο εάν έχει επιλεγεί ένα έργο. Η επιλογή μιας εργασίας έργου είναι προαιρετική. | Εάν αυτό το πεδίο μείνει κενό, ο διαχειριστής έργου μπορεί να αντιστοιχεί στη γραμμή τιμολογίου πωλητή στο αγορασμένο προϊόν που χρησιμοποιείται σε οποιαδήποτε εργασία του έργου. Εάν η γραμμή τιμολογίου του πωλητή δεν αναφέρεται σε μια γραμμή υπεργολαβίας και αυτό το πεδίο μένει κενό, το πραγματικό κόστος που δημιουργείται από τη γραμμή τιμολογίου του πωλητή δεν θα συνδεθεί με οποιεσδήποτε μη μειωμένες πραγματικές τιμές πωλήσεων. Σε αυτή την περίπτωση, εάν έχει ρυθμιστεί χρέωση βάσει εργασιών, το κόστος δεν θα είναι δυνατό να τιμολογηθεί στον τελικό πελάτη. |
| Επιλογή προϊόντος | Επιλέξτε εάν το προϊόν που τιμολογείται είναι υπάρχον προϊόν από τον κατάλογο ή ένα εγγεγραμμένο προϊόν. | Η προεπιλεγμένη τιμή καταχωρείται από τη γραμμή υπεργολαβίας όταν επιλέγεται μια γραμμή υπεργολαβίας. |
| Προϊόν | Επιλέξτε το προϊόν από τον κατάλογο. Εάν το προϊόν είναι εγγεγραμμένο προϊόν, πληκτρολογήστε το όνομα του προϊόντος. | Αυτό το πεδίο χρησιμοποιείται για την εισαγωγή προεπιλεγμένων τιμών αγοράς για υπάρχοντα προϊόντα. |
| Ποσότητα | Εισαγάγετε την ποσότητα του υλικού που τιμολογήθηκε από τον πωλητή σε αυτήν τη γραμμή τιμολογίου. | Κανένα |
| Ομάδα μονάδων | Επιλέξτε την ομάδα μονάδων της μονάδας στην οποία εκφράζονται η ποσότητα που τιμολογήθηκε. | Κανένα |
| Μονάδα | Η προεπιλεγμένη τιμή είναι η βασική μονάδα της ομάδας μονάδων που επιλέγεται. Μπορείτε να αλλάξετε αυτήν την τιμή για να πληρώσετε οποιαδήποτε μονάδα της επιλεγμένης ομάδας μονάδας. | Ο συνδυασμός των τιμών **Προϊόν** και **Μονάδα** θα χρησιμοποιηθεί ως προεπιλεγμένη ή υπολογισμένη τιμή για το πεδίο **Τιμή μονάδας** στη γραμμή τιμολογίου προμηθευτή. |
| Τιμή μονάδας | Η προεπιλεγμένη τιμή μονάδας χρησιμοποιεί τον συνδυασμό των τιμών **Προϊόν** και **μονάδα** από τον τιμοκατάλογο έργου που ισχύει για την ημερομηνία συναλλαγής της γραμμής τιμολογίου πωλητή. | Κανένα |
| Υποσύνολο | Αυτό το πεδίο μόνο για ανάγνωση υπολογίζεται ως *Ποσότητα* &times; *Τιμή μονάδας*, εάν οι τιμές εισάγονται και στο πεδίο **Ποσότητα** και στο πεδίο **Τιμή μονάδας**. Εάν ένα ή και τα δύο αυτά πεδία είναι κενά, μπορείτε να καταχωρήσετε μια τιμή σε αυτό το πεδίο. | Κανένα |
| Φόρος πωλήσεων | Καταχωρήστε το ποσό του φόρου πωλήσεων. | Κανένα |
| Συνολικό ποσό | Το συνολικό ποσό της γραμμής τιμολογίου του πωλητή, συμπεριλαμβανομένων των φόρων. Αυτό το πεδίο υπολογίζεται ως *Υποσύνολο* + *Φόρος πωλήσεων*. | Κανένα |
