# <a name="header-details-for-vendor-invoices"></a>Λεπτομέρειες κεφαλίδας για τιμολόγια προμηθευτών

[!include [banner](../../includes/dataverse-preview.md)]

_**Ισχύει για:** Ελαφριά ανάπτυξη - συμφωνία για προτιμολόγηση_

Αυτό το άρθρο εξηγεί τη λειτουργικότητα που παρέχεται στην κεφαλίδα τιμολογίου πωλητή στο Microsoft Dynamics 365 Project Operations.

Καθώς οι διαχειριστές έργων προγραμματίζουν και εκτελούν έργα, ενδέχεται να απασχολεί υπεργολάβους και να αγοράζουν προϊόντα και υπηρεσίες από προμηθευτές. Κατά την εκτέλεση ενός έργου, το κόστος προκύπτει από τις κατηγορίες υπηρεσιών, υλικού και δαπανών που παρέχονται σε υπεργολάβους με προμηθευτές. Οι προμηθευτές τιμολογούν αυτές τις δαπάνες σε έργα δημιουργώντας τιμολόγια προμηθευτών.

Ο ακόλουθος πίνακας παρέχει πληροφορίες σχετικά με τα πεδία στις κεφαλίδες τιμολογίου προμηθευτή για Project Operations.

| Πεδίο | Description | Αντίκτυπος στη λειτουργία |
| --- | --- | --- |
| 'Ονομα | Το όνομα του τιμολογίου προμηθευτή. | Σε όλες τις αναπτυσσόμενες λίστες τιμολογίου προμηθευτή, το όνομα του τιμολογίου προμηθευτή παρατίθεται στην πρώτη στήλη για να σας βοηθήσει να εντοπίσετε το τιμολόγιο προμηθευτή. Από προεπιλογή, όταν δημιουργείται ένα τιμολόγιο προμηθευτή από υπεργολάβο, το πεδίο **Όνομα** του τιμολογίου προμηθευτή ορίζεται σε μια τιμή που αποτελείται από το όνομα της υπεργολαβίας συν την τρέχουσα ημερομηνία. |
| Description | Μια σύντομη περιγραφή των υπηρεσιών και προϊόντων που τιμολογούνται στο τιμολόγιο προμηθευτή. | Κανένα |
| Πωλητής | Το όνομα της εταιρείας τιμολόγησης για τα προϊόντα και τις υπηρεσίες. Αυτή η εταιρεία πρέπει να είναι μια καρτέλα λογαριασμού που έχει τύπο σχέσης **Πωλητής** ή **Προμηθευτής**. | <p>Με βάση τον επιλεγμένο προμηθευτή, οι προεπιλεγμένες τιμές καταχωρούνται αυτόματα για τα ακόλουθα πεδία:</p><ul><li>Νομισματική μονάδα</li><li>Τιμοκατάλογοι</li><li>Όροι πληρωμής</li><li>Διεύθυνση πληρωμής</li></ul> |
| Υπεργολαβία | Μια αναφορά στην υπεργολαβία για το τιμολόγιο πωλητή. | <p>Με βάση την επιλεγμένη υπεργολαβία, οι προεπιλεγμένες τιμές καταχωρούνται αυτόματα για τα ακόλουθα πεδία:</p><ul><li>Νομισματική μονάδα</li><li>Τιμοκατάλογοι</li><li>Όροι πληρωμής</li><li>Διεύθυνση πληρωμής</li></ul><p>Η υπεργολαβία που επιλέγεται στην κεφαλίδα τιμολογίου πωλητή καταχωρείτε από προεπιλογή στις γραμμές τιμολογίου του πωλητή και δεν μπορεί να αλλάξει εκεί.</p> |
| Ημερομηνία τιμολογίου | Η ημερομηνία για τις πραγματικές τιμές κόστους που θα δημιουργηθούν όταν επιβεβαιωθεί το τιμολόγιο του πωλητή. | Η ημερομηνία τιμολογία επίσης χρησιμοποιείται για την επιλογή του σωστού τιμοκαταλόγου αγοράς είτε από τους τιμοκαταλόγους που επισυνάπτονται στον σχετικό προμηθευτή είτε από τις παραμέτρους του έργου. |
| Αιτιολογία κατάστασης | Η κατάσταση του τιμολογίου πωλητή. | <p>Η κατάσταση καθορίζει τη θέση στην οποία βρίσκεται το τιμολόγιο πωλητή στην επιχειρηματική διεργασία και εάν είναι δυνατή η επεξεργασία της. Ακολουθούν ορισμένες από τις διαθέσιμες τιμές:</p><ul><li>**Προσχέδιο** - Μπορείτε να επεξεργαστείτε το τιμολόγιο πωλητή.</li><li>**Επιβεβαιώθηκε** – Το τιμολόγιο πωλητή επαληθεύτηκε και επιβεβαιώθηκε. Τα τιμολόγια προμηθευτή σε αυτήν την κατάσταση δεν μπορούν να υποστούν επεξεργασία ή να διαγραφούν.</li><li>**Σε εξέλιξη** – Το τιμολόγιο του πωλητή επανεξετάεται. Τα τιμολόγια προμηθευτή σε αυτήν την κατάσταση μπορούν να υποστούν επεξεργασία, αλλά δεν μπορούν να διαγραφούν.</li><li>**Ακυρώθηκε** – Το τιμολόγιο του πωλητή ακυρώθηκε. Τα τιμολόγια προμηθευτή σε αυτήν την κατάσταση δεν μπορούν να υποστούν επεξεργασία ή να διαγραφούν.</li></ul> |
| Νομισματική μονάδα | Η νομισματική μονάδα στην οποία τιμολογεί ο πωλητής για τα προϊόντα και τις υπηρεσίες του τιμολογίου του προμηθευτή. | Σε ένα τιμολόγιο πωλητή που αναφέρεται σε μια υπεργολαβία, η νομισματική μονάδα της υπεργολαβίας καταχωρείται από προεπιλογή ως νομισματική μονάδα του τιμολογίου του πωλητή. Σε ένα τιμολόγιο πωλητή που δεν αναφέρεται σε υπεργολαβία, η προεπιλεγμένη τιμή καταχωρείτε από την καρτέλα λογαριασμού πωλητή και μπορεί να αλλάξει. Μετά την αποθήκευση ενός τιμολογίου πωλητή, δεν είναι πλέον δυνατή η επεξεργασία της νομισματικής μονάδας. |
| Μονάδα συμβάσεων | Η διεύθυνση της εταιρείας που είναι υπεύθυνη για την λήψη των υπηρεσιών ή/και των προϊόντων από τον πωλητή. | Κανένα |
| Όροι πληρωμής | Οι όροι πληρωμής για τιμολόγια προμηθευτών που εκδίδονται. Η προεπιλεγμένη τιμή εισάγεται αυτόματα από την καρτέλα λογαριασμού προμηθευτή. | Οι όροι πληρωμής από μια υπεργολαβία αντιγράφονται σε όλα τα τιμολόγια προμηθευτών που σχετίζονται με την υπεργολαβία. Οι όροι πληρωμής είναι δυνατό να ενημερωθούν εάν η κατάσταση του τιμολογίου προμηθευτή είναι **Προσχέδιο**. |
| Διεύθυνση πληρωμής | Η διεύθυνση του προμηθευτή στην οποία πρέπει να αποστέλλονται οι πληρωμές. Η προεπιλεγμένη τιμή εισάγεται αυτόματα από την καρτέλα λογαριασμού προμηθευτή. | Η διεύθυνση πληρωμής από μια υπεργολαβία αντιγράφεται ως διεύθυνση πληρωμής σε όλα τα τιμολόγια προμηθευτή που δημιουργούνται για αυτήν την υπεργολαβία. Η διεύθυνση πληρωμής είναι δυνατό να ενημερωθεί εάν η κατάσταση του τιμολογίου προμηθευτή είναι **Προσχέδιο**. |
| Υποσύνολο | Εάν ένα τιμολόγιο προμηθευτή δεν έχει γραμμές, εισαγάγετε το υποσύνολο τιμολογίου πριν από τους φόρους. Εάν το τιμολόγιο πωλητή έχει γραμμές, αυτό το πεδίο είναι μόνο για ανάγνωση. Σε αυτήν την περίπτωση, το ποσό που εμφανίζεται είναι το υποσύνολο από όλες τις γραμμές στο τιμολόγιο προμηθευτή. | Κανένα |
| Συνολικός φόρος | Εάν ένα τιμολόγιο προμηθευτή δεν έχει γραμμές, εισαγάγετε τους συνολικούς φόρους στην υπεργολαβία. Εάν το τιμολόγιο πωλητή έχει γραμμές, αυτό το πεδίο είναι μόνο για ανάγνωση. Σε αυτήν την περίπτωση, το ποσό που εμφανίζεται είναι το άθροισμα των ποσών φόρου από όλες τις γραμμές στο τιμολόγιο προμηθευτή. | Κανένα |
| Συνολικό ποσό | Αυτό το υπολογιζόμενο πεδίο εμφανίζει το συνολικό ποσό του τιμολογίου πωλητή μετά την προσθήκη φόρων. | Κανένα |

> [!NOTE]
> Τα ακόλουθα πεδία σε ένα τιμολόγιο πωλητή δεν είναι δυνατό να αλλάξουν μετά την αποθήκευση του τιμολογίου πωλητή: **Πωλητής**, **Υπεργολαβία**, **Νομισματική μονάδα**, **Αναθέτουσα μονάδα** και **Όροι πληρωμής**. Εάν απαιτούνται αλλαγές σε αυτά τα πεδία σε ένα τιμολόγιο πωλητή, πρέπει να διαγράψετε το τιμολόγιο του πωλητή και να δημιουργήσετε ένα νέο τιμολόγιο πωλητή.
