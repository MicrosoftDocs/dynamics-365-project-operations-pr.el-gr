# <a name="subcontract-lines-for-products"></a>Γραμμές για προϊόντα στην υπεργολαβία

[!include [banner](../../includes/dataverse-preview.md)]

_**Ισχύει για:** Ελαφριά ανάπτυξη - συμφωνία για προτιμολόγηση_

Μια υπεργολαβία στο Dynamics 365 Project Operations μπορεί να έχει μια γραμμή υπεργολαβίας για προϊόντα. Αυτές οι γραμμές επιτρέπουν σε έναν υπεύθυνο έργου να αγοράσει προϊόντα από προμηθευτές, τα οποία μπορούν έπειτα να χρησιμοποιήσουν σε εργασίες έργου.

Ολοκληρώστε τα ακόλουθα βήματα για να δημιουργήσετε μια γραμμή υπεργολαβίας για προϊόντα στο Project Operations.

1. Στη σελίδα περιήγησης, επιλέξτε **Υπεργολαβίες** και έπειτα ανοίξτε την υπεργολαβία με την οποία θέλετε να εργαστείτε. 
2. Στην καρτέλα **Γραμμές υπεργολαβίας**, επιλέξτε **+ Δημιουργία** για να δημιουργήσετε μια νέα γραμμή υπεργολαβίας.
3. Στη σελίδα **Γρήγορη δημιουργία**, στο πεδίο **Κλάση συναλλαγής**, επιλέξτε **Υλικό** και εισαγάγετε ή επιλέξτε οποιεσδήποτε τις απαιτούμενες πληροφορίες πεδίου. 
4. Επιλέξτε **Αποθήκευση**.

Στον ακόλουθο πίνακα παρέχονται πληροφορίες σχετικά με τα πεδία στη σελίδα **Λεπτομέρειες γραμμής υπεργολαβίας** και στη σελίδα **Γρήγορη δημιουργία** καθώς σχετίζονται για την αγορά προϊόντων.

| Πεδίο | Περιγραφή |
| ----- | ----------- |
| Ονομασία | Το όνομα της γραμμής υπεργολαβίας. |
| Περιγραφή | Μια σύντομη περιγραφή προϊόντων που παραγγέλνονται στη γραμμη υπεργολαβίας. |
| Τύπος γραμμής | Η τιμή αυτού του πεδίου ορίζεται ως προεπιλογή σε **Βάσει ποσότητας**. |
| Μέθοδος χρέωσης |  Η μέθοδος χρέωσης της γραμμής υπεργολαβίας. Ένα χρονοδιάγραμμα τιμολογίων βάσει ορόσημου είναι διαθέσιμο για μεθόδους χρέωσης προκαθορισμένης τιμής. |
| Κλάση συναλλαγής | Η τιμή αυτού του πεδίου ορίζεται ως προεπιλογή σε **Χρόνος**. Για να δημιουργήσετε γραμμές υπεργολαβίας για την αγορά προϊόντων, στο πεδίο **Κλάση συναλλαγής** επιλέξτε **Υλικό**. Αυτή η επιλογή υποδεικνύει ότι η γραμμή υπεργολαβίας χρησιμοποιείται για την καταγραφή μιας αγοράς προϊόντων για χρήση σε έργα. |
| Επιλογή προϊόντος | Επιλέξτε αν το προϊόν που αγοράζεται διατηρείται στον κατάλογο προϊόντων ή είναι ένα προστιθέμενο προϊόν. |
| Προϊόν | Επιλέξτε ένα ενεργό προϊόν από τον κατάλογο. Αυτό το πεδίο είναι διαθέσιμο μόνο όταν η **Επιλογή προϊόντος** έχει οριστεί σε **Υπάρχον**. |
| Προστεθειμένο προϊόν | Εισαγάγετε το όνομα του προστιθέμενου προϊόντος. Αυτό το πεδίο είναι διαθέσιμο μόνο όταν η **Επιλογή προϊόντος** έχει οριστεί σε **Προστιθέμενο**.  |
| Ημερομηνία παράδοσης που ζητήθηκε | Επιλέξτε την απαιτούμενη ημερομηνία παράδοσης για τα προϊόντα. Αυτή η ημερομηνία χρησιμοποιείται επίσης για την επιλογή ενός τιμοκαταλόγου έργου από τους τιμοκαταλόγους έργου που έχουν επισυναφθεί στην υπεργολαβία. Το κόστος του προϊόντος στη γραμμή υπεργολαβίας ορίζεται έπειτα ως προεπιλογή από αυτόν τον τιμοκατάλογο. |
| Ημερομηνία παράδοσης βάσει σύμβασης | Επιλέξτε την ημερομηνία κατά την οποία τα προϊόντα συμφωνείται με τη σύμβαση για παράδοση.  |
| Παραγγελθείσα ποσότητα | Εισαγάγετε την ποσότητα του προϊόντος που αγοράζεται από τον πωλητή. Εάν ένας υπεύθυνος έργου υπερβεί αυτήν την ποσότητα, εμφανίζεται μια προειδοποίηση. |
| Ομάδα μονάδων | Οι προεπιλογές αυτής της τιμής μόνο για προϊόντα καταλόγου. Όταν είναι επιλεγμένα και τα δύο **Προϊόν** και **Ημερομηνία παράδοσης που ζητήθηκε**, το σύστημα επιλέγει τον ισχύοντα τιμοκατάλογο με βάση την ημερομηνία παράδοσης. Τα σχετικά στοιχεία τιμοκαταλόγου ερωτώνται για το προϊόν που ταιριάζει. Οι τιμές μονάδας και ομάδας μονάδων ορίζονται ως προεπιλογή από τη ρύθμιση στην καρτέλα στοιχείου τιμοκκαταλόγου. |
| Μονάδα | Αυτή η τιμή ορίζεται ως προεπιλογή στη ρύθμιση μονάδας στην καρτέλα στοιχείου τιμοκαταλόγου. Μπορείτε να αλλάξετε αυτήν τη μονάδα σε άλλη μονάδα, εάν χρειάζεται. Ο συνδυασμός προϊόντος και μονάδας χρησιμοποιείται για τον ορισμό ως προεπιλογή της τιμής μονάδας στη γραμμή υπεργολαβίας για υπάρχοντα προϊόντα καταλόγου. |
| Τιμή μονάδας | Η τιμή μονάδας ορίζεται ως προεπιλογή χρησιμοποιώντας τον συνδυασμό προϊόντος και μονάδας από τα στοιχεία τιμοκαταλογου που σχετίζονται με τον τιμοκατάλογο έργου που ισχύει για τη ζητηθείσα ημερομηνία παράδοσης της γραμμής υπεργολαβίας.  |
| Υποσύνολο | Αυτό το πεδίο μόνο για ανάγνωση υπολογίζεται ως Ποσότητα x τιμή μονάδας εάν και τα δύο πεδία έχουν τιμές που έχουν καταχωρηθεί. Εάν οποιοαδήποτε από τα πεδία **Ποσότητα** ή **Τιμή μονάδας** ή και τα δύο είναι κενά, μπορείτε να εισαγάγετε μια τιμή με μη αυτόματο τρόπο.  |
| Φόρος πωλήσεων | Καταχωρήστε την τιμή του φόρου πωλήσεων. |
| Συνολικό ποσό | Αυτό το πεδίο υπολογισμού εμφανίζει το συνολικό ποσό της γραμμής υπεργολαβίας αφού συμπεριληφθούν φόροι. Η τιμή σε αυτό το πεδίο υπολογίζεται ως υποσύνολο + φόρος. |
