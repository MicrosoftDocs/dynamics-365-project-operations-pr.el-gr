# <a name="vendor-invoice-lines-for-expense-categories"></a>Γραμμές τιμολογίου προμηθευτή για κατηγορίες εξόδων

[!include [banner](../../includes/dataverse-preview.md)]

_**Ισχύει για:** Ελαφριά ανάπτυξη - συμφωνία για προτιμολόγηση_

Ένα τιμολόγιο πωλητή στο Microsoft Dynamics 365 Project Operations μπορεί να έχει γραμμές τιμολογίου πωλητή για τις κατηγορίες δαπανών. Οι διαχειριστές έργου μπορούν να χρησιμοποιήσουν γραμμές τιμολογίου πωλητή για τις κατηγορίες δαπανών για να καταγράψουν το κόστος των υπηρεσιών που παρέχονται ως κατηγορίες δαπανών.

Οι γραμμές τιμολογίου του προμηθευτή για κατηγορίες δαπανών ενδέχεται να αναφέρονται σε μια γραμμή υπεργολαβίας για κατηγορίες δαπανών ή όχι. Εάν μια γραμμή τιμολογίου πωλητή για κατηγορίες δαπανών αναφέρεται σε υπεργολαβία, οι διαχειριστές έργου θα μπορούν να αντιστοιχιστούν και να επαληθεύσουν τις δαπάνες που τιμολογούνται από τη γραμμή τιμολογίου του πωλητή με βάση τις δαπάνες που καταγράφονται σε αυτές τις κατηγορίες δαπανών και έχουν εγκριθεί από τους διαχειριστές έργου στο έργο.

Ο ακόλουθος πίνακας παρέχει πληροφορίες σχετικά με τα πεδία στις γραμμές τιμολογίου προμηθευτή για κατηγορίες δαπανών.

| Πεδίο | Description | Αντίκτυπος στη λειτουργία |
| --- | --- | --- |
| 'Ονομα | Το όνομα της γραμμής τιμολογίου πωλητή που θα βοηθήσει με την αναγνώριση. | Αυτό το όνομα θα εμφανίζεται ως η πρώτη στήλη σε όλες τις αναζητήσεις που βασίζονται σε γραμμές τιμολογίου πωλητή. |
| Description | Μια σύντομη περιγραφή των υπηρεσιών που τιμολογούνται από τον πωλητή στη γραμμή τιμολογίου του πωλητή. | Κανένα |
| Υπεργολαβία | Η υπεργολαβία στην οποία παραγγέλθηκαν αρχικά οι υπηρεσίες. | Όταν επιλέγεται μια υπεργολαβία για το τιμολόγιο πωλητή, σε όλες τις γραμμές του τιμολογίου πωλητή θα μεταβιβάζεται αυτή η επιλογή. Ένα τιμολόγιο πωλητή δεν μπορεί να έχει γραμμές τιμολογίου πωλητή που να αναφέρονται σε διαφορετικές υπεργολαβίες. |
| Γραμμή υπεργολαβίας | Η γραμμή υπεργολαβίας στην οποία παραγγέλθηκαν οι υπηρεσίες. Η λίστα των γραμμών υπεργολαβίας που μπορούν να επιλεγούν περιορίζεται στις γραμμές στην επιλεγμένη υπεργολαβία. | Όταν επιλέγεται μια γραμμή υπεργολαβίας σε μια γραμμή τιμολογίου πωλητή για κατηγορίες δαπανών, εισάγονται από τα αντίστοιχα πεδία της γραμμής υπεργολαβίας προεπιλεγμένες τιμές για τα πεδία **Έργο**, **Εργασία** και **Κατηγορία συναλλαγής**. Εάν η επιλεγμένη γραμμή υπεργολαβίας έχει τιμές στα πεδία **Έργο**, **Εργασία έργου** και **Κατηγορία συναλλαγής**, οι τιμές των αντίστοιχων πεδίων στη γραμμή τιμολογίου του πωλητή δεν μπορούν να διαφέρουν από αυτές τις τιμές. |
| Ημερομηνία συναλλαγής | Η ημερομηνία κατά την οποία το πραγματικό κόστος της γραμμής τιμολογίου πωλητή θα καταγραφεί στο έργο. |Κανένα |
| Κλάση συναλλαγής | Επιλέξτε **Δαπάνες** για να καταγράψετε ένα τιμολόγιο πωλητή για μια κατηγορία δαπανών. | Η τιμή **Δαπάνες** δηλώνει ότι η γραμμή τιμολογίου του προμηθευτή χρησιμοποιείται για την καταγραφή του ποσού τιμολογίου για τις υπηρεσίες που είχαν προμηθευθεί ως κατηγορίες δαπανών. |
| Project | Το όνομα του έργου στο οποίο χρησιμοποιήθηκαν οι υπηρεσίες που τιμολογήθηκαν. | Αυτό το πεδίο είναι υποχρεωτικό και δεν είναι δυνατό να μείνει κενό. |
| Κλείσιμο εργασίας | Το όνομα της εργασίας έργου στο οποίο χρησιμοποιήθηκαν οι υπηρεσίες που τιμολογήθηκαν. Αυτό το πεδίο είναι διαθέσιμο μόνο εάν έχει επιλεγεί ένα έργο. Η επιλογή μιας εργασίας έργου είναι προαιρετική. | Εάν αυτό το πεδίο μείνει κενό, ο διαχειριστής έργου μπορεί να αντιστοιχεί στη γραμμή τιμολογίου πωλητή σε έξοδα που καταγράφεται σε οποιαδήποτε εργασία του έργου. Εάν η γραμμή τιμολογίου του πωλητή δεν αναφέρεται σε μια γραμμή υπεργολαβίας και αυτό το πεδίο μένει κενό, το πραγματικό κόστος που δημιουργείται από τη γραμμή τιμολογίου του πωλητή δεν θα συνδεθεί με οποιεσδήποτε μη μειωμένες πραγματικές τιμές πωλήσεων. Σε αυτή την περίπτωση, εάν έχει ρυθμιστεί χρέωση βάσει εργασιών, το κόστος ενδεχομένως να μην είναι δυνατό να τιμολογηθεί στον τελικό πελάτη. |
| Κατηγορία συναλλαγής | Η κατηγορία συναλλαγής που τιμολογείτε. Για την επιλεγμένη κατηγορία συναλλαγής πρέπει να δημιουργηθεί μια αντίστοιχη κατηγορία δαπανών. | Ο συνδυασμός των τιμών **Κατηγορία συναλλαγών** και **Μονάδα** θα χρησιμοποιηθεί ως προεπιλεγμένη ή υπολογισμένη τιμή για το πεδίο **Τιμή μονάδας** στη γραμμή τιμολογίου προμηθευτή. |
| Ποσότητα | Εισαγάγετε την ποσότητα που τιμολογήθηκε από τον πωλητή στη γραμμή τιμολογίου. |Κανένα|
| Ομάδα μονάδων | Μια προεπιλεγμένη τιμή εισάγεται με βάση την ομάδα μονάδων της επιλεγμένης κατηγορίας συναλλαγής. | Κανένα |
| Μονάδα | Η προεπιλεγμένη τιμή είναι η βασική μονάδα της ομάδας μονάδων που επιλέγεται. Μπορείτε να αλλάξετε αυτήν την τιμή για να αγοράσετε οποιαδήποτε μονάδα της Ομάδας μονάδας. | Ο συνδυασμός των τιμών **Κατηγορία συναλλαγών** και **Μονάδα** θα χρησιμοποιηθεί ως προεπιλεγμένη ή υπολογισμένη τιμή για το πεδίο **Τιμή μονάδας** στη γραμμή τιμολογίου προμηθευτή. |
| Τιμή μονάδας | Η προεπιλεγμένη τιμή μονάδας χρησιμοποιεί τον συνδυασμό των τιμών **κατηγορία συναλλαγής** και **μονάδα** από τον τιμοκατάλογο έργου που ισχύει για την ημερομηνία συναλλαγής της γραμμής τιμολογίου πωλητή. | Εάν η τιμή για τον ισχύοντα τιμοκατάλογο έργου έχει οριστεί σε μια μονάδα που διαφέρει από τη μονάδα στη γραμμή τιμολογίου του πωλητή, το σύστημα χρησιμοποιεί τη μετατροπή μονάδας για τον υπολογισμό της τιμής ανά μονάδα. |
| Υποσύνολο | Αυτό το πεδίο μόνο για ανάγνωση υπολογίζεται ως *Ποσότητα* &times; *Τιμή μονάδας*, εάν οι τιμές εισάγονται και στο πεδίο **Ποσότητα** και στο πεδίο **Τιμή μονάδας**. Εάν ένα ή και τα δύο αυτά πεδία είναι κενά, μπορείτε να καταχωρήσετε μια τιμή σε αυτό το πεδίο.| Κανένα |
| Φόρος πωλήσεων | Καταχωρήστε το ποσό του φόρου πωλήσεων. | Κανένα |
| Συνολικό ποσό | Το συνολικό ποσό της γραμμής τιμολογίου του πωλητή, συμπεριλαμβανομένων των φόρων. Αυτό το πεδίο υπολογίζεται ως *Υποσύνολο* + *Φόρος πωλήσεων*. | Κανένα |
