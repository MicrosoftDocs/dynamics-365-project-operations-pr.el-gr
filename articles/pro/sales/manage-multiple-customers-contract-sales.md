# <a name="manage-multiple-customers-on-project-contracts---lite"></a>Διαχείριση πολλών πελατών σε συμβάσεις έργου - lite

_**Ισχύει για:** Ελαφριά ανάπτυξη - συμφωνία για προτιμολόγηση_

Οι συμβάσεις έργου στο Dynamics 365 Project Operations υποστηρίζουν το σενάριο όπου μια συμβατική συμφωνία περιλαμβάνει πολλούς πελάτες που χρηματοδοτούν μια συμφωνία. Η καρτέλα **Σύνοψη** στη σελίδα **Σύμβαση έργου** περιλαμβάνει το πεδίο **Πελάτης**. Αυτό το πεδίο προσδιορίζει τον κύριο πελάτη της συμφωνίας. Είναι δυνατή η ρύθμιση άλλων πελατών για τη συμφωνία στην καρτέλα **Πελάτες** της σελίδας **Σύμβαση έργου**.

Όλοι οι πελάτες σύμβασης στη σελίδα Πελάτες της προεπιλεγμένης σύμβασης έργου ως πελάτες γραμμής σύμβασης σε τυχόν νέες γραμμές σύμβασης βάσει έργου που δημιουργήθηκαν για τη σύμβαση έργου. Οι υπάρχουσες γραμμές σύμβασης βάσει έργου δεν μεταβιβάζουν νέους πελάτες σύμβασης καθώς δημιουργούνται νέες καρτέλες.

Οι γραμμές σύμβασης βάσει έργου συσχετίζονται αυτόματα με τον κύριο πελάτη.

Οι πελάτες σύμβασης και οι πελάτες γραμμής σύμβασης μπορούν να προστεθούν, να ενημερωθούν ή να διαγραφούν οποιαδήποτε στιγμή προτού κερδηθεί η σύμβαση.

## <a name="primary-customer"></a>Κύριος πελάτης

Ο πελάτης που περιλαμβάνεται στην καρτέλα **Σύνοψη** της σύμβασης έργου καθώς ο πιθανός πελάτης είναι ο κύριος πελάτης της σύμβασης. Όταν προσπαθείτε να διαγράψετε τον κύριο πελάτη από τη λίστα πελατών της σύμβασης, θα λάβετε ένα μήνυμα σφάλματος ότι δεν είναι δυνατή η διαγραφή μιας κύριας καρτέλας πελάτη σε μια σύμβαση.

Ο κύριος πελάτης δεν μπορεί να ενημερωθεί από τη λίστα "πελάτες σύμβασης". Αντί για αυτό, αλλάξτε τον πιθανό πελάτη στην καρτέλα **Σύνοψη** της σύμβασης. Όταν αυτό το πεδίο ενημερώνεται στη σελίδα **Σύνοψη σύμβασης**, ο νέος πελάτης προστίθεται ως νέος πελάτης σύμβασης με τη σημαία **Κύριος**. Ο προηγούμενος κύριος πελάτης θα εξακολουθεί να είναι πελάτης της σύμβασης.

## <a name="edit-billing-split-percentages"></a>Επεξεργασία ποσοστών διαίρεσης χρέωσης

Είναι δυνατή η επεξεργασία των ποσοστών διαίρεσης χρέωσης χρησιμοποιώντας τη δυνατότητα επεξεργασίας που είναι ενσωματωμένη στο πλέγμα. Όταν τα ποσοστά διαίρεσης χρέωσης δεν είναι συνολικά 100 τοις εκατό, θα εμφανιστεί ένα σφάλμα. Αφού επεξεργαστείτε τα ποσοστά διαίρεσης χρέωσης, ανανεώστε τη σελίδα για να καταργήσετε το σφάλμα.

Μπορείτε, επίσης, να επιλέξετε **Ομοιόμορφη κατανομή** στο υποπλέγμα **Πελάτες σύμβασης** για να κατανείμετε ομοιόμορφα τις διαιρέσεις τιμολόγησς σε όλους τους πελάτες της σύμβασης. Εάν υπάρχει συντελεστής στρογγυλοποίησης, θα προστεθεί στον πελάτη στρογγυλοποίησης. Ένας από τους πελάτες της σύμβασης είναι πάντα χαρακτηρισμένος ως πελάτης **στρογγυλοποίησης**, γεγονός που σημαίνει ότι η καρτέλα του πελάτη σύμβασης έχει τη σημαία στρογγυλοποίησης ρυθμισμένη στο **Ναι**. Κατά κανόνα, αυτός είναι ο κύριος πελάτης της σύμβασης, αλλά μπορεί επίσης να αλλάξει.
