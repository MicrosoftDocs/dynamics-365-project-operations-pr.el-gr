# <a name="estimating-a-project-based-quote-line"></a>Εκτίμηση μιας γραμμής προσφοράς βάσει έργου

_**Ισχύει για:** Εργασίες έργου για σενάρια βασισμένα σε πόρους/μη εφοδιασμένα, ανάπτυξη Lite - συμφωνία για προτιμολόγηση_

Μια γραμμή προσφοράς βάσει έργου περιέχει λεπτομέρειες που σας βοηθούν να υπολογίσετε το κόστος και τα πιθανά έσοδα της εργασίας που σχετίζεται με την παράδοση της γραμμής προσφοράς.

Για να υπολογίσετε μια γραμμή προσφοράς βάσει έργου, στη γραμμή προσφοράς βάσει έργου, επιλέξτε την καρτέλα **Λεπτομέρειες γραμμής προσφοράς**. Υπάρχουν δύο τρόποι για να δημιουργήσετε μια εκτίμηση σε μια γραμμή προσφοράς βάσει έργου:

- Δημιουργήστε με μη αυτόματο τρόπο την εκτίμηση απευθείας στη γραμμή προσφοράς χρησιμοποιώντας τις λεπτομέρειες της γραμμής προσφοράς 
- Δημιουργήστε ένα έργο και ένα σχέδιο έργου και, στη συνέχεια, συσχετίστε το έργο και τις εργασίες του έργου με τη γραμμή προσφοράς. Θα ενεργοποιηθεί η διεργασία εισαγωγής των προβλέψεων στο πλάνο έργου στη γραμμή προσφοράς με βάση τις πληροφορίες που παρείχατε.

## <a name="create-estimates-directly-on-a-project-based-quote-line"></a>Δημιουργία εκτιμήσεων απευθείας για μια γραμμή προσφοράς βάσει έργου

Για να δημιουργήσετε μια εκτίμηση σε μια γραμμή προσφοράς βάσει έργου, επιλέξτε την καρτέλα **Λεπτομέρεια γραμμής προσφοράς**. Το στοιχείο γραμμής που δημιουργείτε σε αυτήν την καρτέλα θα συνοψίσει την τιμή της προσφοράς για αυτήν τη γραμμή προσφοράς. 

Για να δημιουργήσετε λεπτομέρειες για τη γραμμή προσφοράς, επιλέξτε **+ Νέα λεπτομέρεια γραμμής προσφοράς** στο υποπλέγμα **Λεπτομέρειες γραμμής προσφοράς**. Θα ανοίξει ένα ρυθμιστικό γρήγορης δημιουργίας. Τα ακόλουθα πεδία στη φόρμα **Γραμμή προσφοράς**:

| **Πεδίο** | **Θέση** | **Περιγραφή** | **Κατάντη επίπτωση** |
| --- | --- | --- | --- |
| Περιγραφή | Γρήγορη δημιουργία | Μια περιγραφή της συγκεκριμένης εκτίμησης. | Αυτό το πεδίο προεπιλέγεται στη σχετική λεπτομέρεια της γραμμής προσφοράς για το κόστος που δημιουργείται αυτόματα. |
| Κλάση συναλλαγής | Γρήγορη δημιουργία | Αυτή η αναπτυσσόμενη λίστα παρέχει τις κλάσεις συναλλαγής που περιλαμβάνονται στην καρτέλα **Γενικά** της γραμμής προσφοράς βάσει έργου.  | Αυτό το πεδίο προεπιλέγεται στη σχετική λεπτομέρεια της γραμμής προσφοράς για το κόστος που δημιουργείται αυτόματα. |
| Ρόλος | Γρήγορη δημιουργία | Το άτομο που θα εκτελέσει αυτήν την εργασία ή θα επιβαρυνθεί με αυτήν τη δαπάνη. | Αυτό το πεδίο προεπιλέγεται στη σχετική λεπτομέρεια της γραμμής προσφοράς για το κόστος που δημιουργείται αυτόματα. |
| Κατηγορία | Γρήγορη δημιουργία | Κατηγορία της εργασίας ή του εξόδου. | Αυτό το πεδίο προεπιλέγεται στη σχετική λεπτομέρεια της γραμμής προσφοράς για το κόστος που δημιουργείται αυτόματα. |
| Ημερομηνία έναρξης | Γρήγορη δημιουργία | Ημερομηνία έναρξης της εργασίας. | Αυτό το πεδίο προεπιλέγεται στη σχετική λεπτομέρεια της γραμμής προσφοράς για το κόστος που δημιουργείται αυτόματα. |
| Ημερομηνία λήξης | Γρήγορη δημιουργία | Ημερομηνία λήξης της εργασίας. | Αυτό το πεδίο προεπιλέγεται στη σχετική λεπτομέρεια της γραμμής προσφοράς για το κόστος που δημιουργείται αυτόματα. |
| Μονάδα πόρων | Γρήγορη δημιουργία | Μονάδα πόρων που θα επωμιστεί αυτό το κόστος και θα παρέχει τον πόρο που θα εργαστεί ως προς αυτό. | Αυτό το πεδίο προεπιλέγεται στη σχετική λεπτομέρεια της γραμμής προσφοράς για το κόστος που δημιουργείται αυτόματα. Αυτό το πεδίο χρησιμοποιείται επίσης στην ανάκτηση της τιμής κόστους. |
| Χρονοδιάγραμμα μονάδας | Γρήγορη δημιουργία | Ομάδα μονάδων του έργου ή της δαπάνης. Οι μονάδες ανήκουν σε ένα χρονοδιάγραμμα μονάδων ή σε μια ομάδα μονάδων. Για παράδειγμα, τα μίλια και τα χιλιόμετρα είναι μονάδες που ανήκουν σε μια ομάδα μονάδων που περιγράφουν την απόσταση. | Αυτό το πεδίο προεπιλέγεται στη σχετική λεπτομέρεια της γραμμής προσφοράς για το κόστος που δημιουργείται αυτόματα. |
| Μονάδα | Γρήγορη δημιουργία | Μονάδα εργασίας ή δαπάνης. | Αυτό το πεδίο προεπιλέγεται στη σχετική λεπτομέρεια της γραμμής προσφοράς για το κόστος που δημιουργείται αυτόματα. |
| Ποσότητα | Γρήγορη δημιουργία | Ποσότητα εργασίας ή δαπάνης | Αυτό το πεδίο προεπιλέγεται στη σχετική λεπτομέρεια της γραμμής προσφοράς για το κόστος που δημιουργείται αυτόματα. |

Αυτό το πεδίο προκαθορίζεται από προεπιλογή για το χρόνο, βάσει του ρόλου και του συνδυασμού της τιμής της μονάδας σύμβασης του τιμοκαταλόγου προσφοράς ο οποίος ισχύει για την ημερομηνία έναρξης. Για έξοδα, αυτό το πεδίο προκαθορίζεται από την παραμετροποίηση της τιμής για την κατηγορία συναλλαγής στον τιμοκατάλογο κόστους της μονάδας σύμβασης που ισχύει για την ημερομηνία έναρξης. Εάν η μέθοδος τιμολόγησης για την κατηγορία συναλλαγής δεν είναι "τιμή ανά μονάδα", δεν υπάρχει καμία προεπιλογή και αυτό το πεδίο παραμένει κενό. |

| Ποσό | Γρήγορη δημιουργία | Μπορείτε να εισαγάγετε πληροφορίες με μη αυτόματο τρόπο σε αυτό το πεδίο, εάν τα πεδία **Ποσότητα** και **Τιμή** παραμείνουν κενά. Εάν αυτά τα πεδία δεν είναι κενά, αυτό το πεδίο γίνεται μόνο για ανάγνωση και υπολογίζεται ως (Ποσότητα \* Τιμή μονάδας) + Φόρος. | Δεν υπάρχει καμία κατάντη επίπτωση αυτού του πεδίου. |

## <a name="update-prices-on-quote-line-details"></a>Ενημέρωση τιμών στις λεπτομέρειες της γραμμής προσφοράς

Εάν έχετε αλλάξει τις τιμές στον τιμοκατάλογο έργου που επισυνάπτεται στην προσφορά ή στον τιμοκατάλογο της σύμβασης, μπορείτε να επιλέξετε **Νέος υπολογισμός** στη σελίδα **Προσφορά**, για να ανανεώσετε τις τιμές στις λεπτομέρειες της συγκεκριμένης γραμμής προσφοράς ώστε να αντικατοπτρίζεται αυτή η αλλαγή. Όταν επιλέγετε **Εκ νέου υπολογισμός**, εμφανίζεται μια προειδοποίηση που σας ενημερώνει ότι θα γίνει επαναφορά των τιμών στις λεπτομέρειες της γραμμής προσφοράς για όλες τις γραμμές προσφοράς σε αυτήν την προσφορά. Επιλέξτε **Ναι** για να ανανεώσετε τις τιμές τόσο για τις λεπτομέρειες της προσφοράς όσο και για τις γραμμές προσφοράς κόστους.

## <a name="access-quote-line-details-for-cost"></a>Πρόσβαση σε λεπτομέρειες της γραμμής προσφοράς για το κόστος

Στην καρτέλα **Λεπτομέρειες γραμμής προσφοράς**, επιλέξτε μια γραμμή στο πλέγμα για να ενεργοποιήσετε ορισμένες ενέργειες στη γραμμή εργαλείων του υποπλέγματος. Η πρώτη ενέργεια στη γραμμή εργαλείων υποπλέγματος όταν έχει επιλεγεί μια λεπτομέρεια γραμμής προσφοράς είναι **Άνοιγμα λεπτομέρειας κόστους**. Επιλέξτε **Άνοιγμα λεπτομέρειας κόστους** για να δείτε τη σχετική χρέωση κόστους και το ποσό για αυτήν τη γραμμή προσφοράς.

> [!NOTE]
> Η αλλαγή των τιμών της μονάδας, της ποσότητας, των ημερομηνιών, του ρόλου ή της κατηγορίας στη γραμμή προσφοράς για το κόστος θα αλλάξει τις αντίστοιχες τιμές στις λεπτομέρειες της γραμμής προσφοράς για τις πωλήσεις.
## <a name="currency-on-quote-line-details-for-cost-and-sales"></a>Νόμισμα στις λεπτομέρειες της γραμμής προσφοράς για το κόστος και τις πωλήσεις

Νομισματική μονάδα στη λεπτομέρεια της γραμμής προσφοράς για τις προεπιλογές πωλήσεων από τον τιμοκατάλογο έργου που ισχύει για την ημερομηνία έναρξης της λεπτομέρειας της προσφοράς.

Νομισματική μονάδα στη λεπτομέρεια της γραμμής προσφοράς για τις προεπιλογές κόστους από τον τιμοκατάλογο της συμβατικής μονάδας της προσφοράς που ισχύει για την ημερομηνία έναρξης της λεπτομέρειας της γραμμής προσφοράς για το κόστος.

Οι υπολογισμοί κερδοφορίας μετατρέπουν το ποσό στις λεπτομέρειες της γραμμής προσφοράς για το κόστος και τις πωλήσεις στη βασική νομισματική μονάδα του περιβάλλοντος για την αναφορά του συνολικού εκτιμώμενου περιθωρίου στην προσφορά.

Αυτό μπορεί να οδηγήσει σε σφάλματα στρογγυλοποίησης νομισματικών μονάδων και σε μεταβαλλόμενα περιθώρια λόγω έλλειψης ημερομηνίας αποδοτικής συναλλαγματικής ισοτιμίας. Χρησιμοποιήστε αυτούς τους υπολογισμούς σε προσφορές έργου μόνο ως προσεγγίσεις και όχι ως πραγματικές νομικές ή άλλες αναφορές που απαιτούν μεγαλύτερη ακρίβεια όσον αφορά τη στρογγυλοποίηση και την ενημέρωση σχετικά με την ημερομηνία ισχύος των συναλλαγματικών ισοτιμιών.
