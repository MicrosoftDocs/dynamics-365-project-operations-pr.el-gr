# <a name="manage-a-proforma-project-invoice"></a>Διαχείριση προτιμολογίου έργου 

_**Ισχύει για:** Ελαφριά ανάπτυξη - συμφωνία για προτιμολόγηση_

Στο Dynamics 365 Project Operations, τα προτιμολόγια δημιουργούνται ως επέκταση των τιμολογίων στο Dynamics 365 Sales. Ωστόσο, υπάρχουν πολλές διαφορές στη διαδικασία τιμολόγησης μεταξύ του Sales και του Project Operations όσον αφορά την τιμολόγηση. Για παράδειγμα, δεν είναι δυνατό να δημιουργήσετε ένα νέο τιμολόγιο από τη σελίδα **Λίστα τιμολογίων** στο Project Operations αλλά μπορείτε να το κάνετε στο Sales. Αυτές οι διαφορές και οι επεκτάσεις έχουν γίνει για την υποστήριξη διεργασιών τιμολόγησης για έργα που διαφέρουν από ένα τυπικό τιμολόγιο για μια παραγγελία πώλησης.

> [!IMPORTANT]
> Λόγω των διαφορών, μην χρησιμοποιείτε εναλλακτικά τα τιμολόγια στο Sales και το Project Operations.

## <a name="invoice-line-details"></a>Λεπτομέρειες γραμμής τιμολογίου

Κάθε γραμμή τιμολογίου σε ένα τιμολόγιο έργου περιλαμβάνει λεπτομέρειες γραμμής τιμολογίου. Αυτές οι λεπτομέρειες γραμμής σχετίζονται με τις πραγματικές μη χρεωμένες πωλήσεις και τα ορόσημα που έχουν σχέση με τη γραμμή σύμβασης η οποία αναφέρεται από τη γραμμή τιμολογίου. Όλες αυτές οι συναλλαγές επισημαίνονται ως **Έτοιμο για τιμολόγηση**.

Για μια γραμμή **τιμολογίου χρόνου και υλικού**, οι λεπτομέρειες γραμμής τιμολογίου ομαδοποιώνται σε **Χρεώσιμο**, **Μη χρεώσιμο** και **Δωρεάν** στη σελίδα **Γραμμή τιμολογίου**. Οι λεπτομέρειες **Γραμμή τιμολογίου με χρέωση** προστίθενται στο σύνολο της γραμμής τιμολογίου. **Οι δωρεάν** και **μη χρεώσιμες πραγματικές τιμές** δεν προστίθενται στο σύνολο της γραμμής τιμολογίου.

Για μια γραμμή **τιμολογίου προκαθορισμένης τιμής**, οι λεπτομέρειες γραμμής τιμολογίου δημιουργούνται από ορόσημα τα οποία έχουν επισημανθεί ως **Έτοιμο για τιμολόγηση** στη σχετική γραμμή σύμβασης. Αφού δημιουργηθεί η λεπτομέρεια της γραμμής τιμολογίου από ένα ορόσημο, η κατάσταση χρέωσης στο ορόσημο ενημερώνεται σε **Δημιουργήθηκε τιμολόγιο πελάτη**.

### <a name="create-new-invoice-line-details"></a>Δημιουργήστε νέες λεπτομέρειες γραμμής τιμολογίου

Σε γραμμές τιμολογίου χρόνου και υλικού, μπορείτε να δημιουργήσετε νέες λεπτομέρειες γραμμής τιμολογίου. Αυτές οι λεπτομέρειες γραμμής τιμολογίου δεν υποστηρίζονται από ένα πραγματικό στοιχείο. Στη γραμμή τιμολογίου μιας γραμμής τιμολογίου χρόνου και υλικού, μπορείτε να επιλέξετε **Δημιουργία** για να δημιουργήσετε μια νέα λεπτομέρεια γραμμής τιμολογίου για τις κατηγορίες συναλλαγών που περιλαμβάνονται στη γραμμή σύμβασης.

## <a name="refresh-invoice-transactions"></a>Ανανέωση συναλλαγών τιμολογίου

Εάν έχετε πραγματικές τιμές που εμφανίστηκαν μετά τη δημιουργία του τιμολογίου, μπορείτε να συμπεριλάβετε αυτές τις πραγματικές τιμές στο τιμολόγιο.

1. Στην **Προβολή εκκρεμών εργασιών χρέωσης**, επισημάνετε τα δεδομένα ως **Έτοιμο για τιμολόγηση**.   
2. Ανοίξτε το προσχέδιο προτιμολογίου και, στην κορδέλα **Ενέργειες**, κάντε κλικ στην επιλογή **Ανανέωση συναλλαγών γραμμής τιμολογίου**.

  Με αυτόν τον τρόπο δημιουργούνται λεπτομέρειες για τη γραμμή τιμολογίου για κάθε πραγματική τιμή που έχει ημερομηνία λήξης και έχει επισημανθεί ως **Έτοιμο για τιμολόγηση** αλλά δεν περιλαμβάνεται στο τιμολόγιο.

## <a name="product-based-invoice-lines"></a>Γραμμές τιμολογίου βάσει προϊόντος

Στο Project Operations, μπορείτε να δημιουργήσετε γραμμές τιμολογίου για προϊόντα που δεν ισχύουν για κανένα έργο ή για όλα τα έργα μαζί με γραμμές τιμολογίου βάσει έργου. Αυτές οι γραμμές τιμολογίου δημιουργούνται ως γραμμές σύμβασης βασισμένες σε προϊόντα και αφού επισημανθούν ως έτοιμες προς τιμολόγηση, προστίθενται ως γραμμές τιμολογίου με βάση προϊόντα.

Αφού προσθέσετε τις γραμμές τιμολογίου που βασίζονται σε προϊόντα, δεν είναι δυνατή η αλλαγή τους. Ωστόσο, είναι δυνατό να διαγραφούν από το προσχέδιο προτιμολογίου.
