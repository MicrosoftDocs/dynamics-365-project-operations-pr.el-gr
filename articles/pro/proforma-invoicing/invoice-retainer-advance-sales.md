# <a name="invoice-a-retainer-or-an-advance"></a>Τιμολόγιο μιας προκαταβολής ή προπληρωμής

_**Ισχύει για:** Εργασίες έργου για σενάρια βασισμένα σε πόρους/μη εφοδιασμένα, ανάπτυξη Lite - συμφωνία για προτιμολόγηση_

Το Dynamics 365 Project Operations υποστηρίζει συμβάσεις βάσει προπληρωμών και άπαξ προκαταβολές. Σε μια σύμβαση έργου, μπορείτε να καταχωρήσετε ένα χρονοδιάγραμμα προκαταβολής ή προπληρωμής. Ωστόσο, η εγγραφή σε επίπεδο σύμβασης έργου δεν καθιστά αμέσως διαθέσιμη για χρήση μια προκαταβολή ή μια προπληρωμή. Για να χρησιμοποιήσετε μια προκαταβολή ή μια προπληρωμή σε ένα τιμολόγιο που πραγματικά χρεώνει τον πελάτη, η προκαταβολή ή η προπληρωμή πρέπει να τιμολογηθεί πρώτα.

Ολοκληρώστε τα παρακάτω βήματα για να τιμολογήσετε μια προκαταβολή ή μια προπληρωμή.

1. Επιλέξτε **Πωλήσεις** > **Τιμολόγηση** > **Προκαταβολές και Προπληρωμές**. 
2. Στη σελίδα **Προκαταβολές και Προπληρωμές**, χρησιμοποιήστε το φίλτρο για να επιλέξετε τη συγκεκριμένη προκαταβολή ή την προπληρωμή για να την τιμολογήσετε και επισημάνετέ την ως **Έτοιμο για τιμολόγηση**.
3. Δημιουργήστε ένα τιμολόγιο είτε με μη αυτόματο τρόπο από τη λίστα **Σύμβαση έργου** ή τη σελίδα λεπτομερειών. Η προκαταβολή ή η προπληρωμή εμφανίζεται στο προσχέδιο τιμολογίου στην ενότητα **Προπληρωμές και Προκαταβολές** στη σελίδα **Τιμολόγιο**.
4. Επιβεβαίωση τιμολογίου. Η προκαταβολή ή η προπληρωμή θα είναι διαθέσιμη για χρήση. Μπορείτε να επαληθεύσετε το τιμολόγιο στη σελίδα λίστας **Προκαταβολές και Προπληρωμές**. Για μια τιμολογημένη προκαταβολή ή μια προπληρωμή, το διαθέσιμο ποσό εμφανίζεται στο πλέγμα.

## <a name="create-a-retainer-or-advance-from-the-invoice-grid"></a>Δημιουργία μιας προκαταβολής ή μιας προπληρωμής από το πλέγμα τιμολογίου

Μπορείτε να δημιουργήσετε μια προκαταβολή ή μια προπληρωμή απευθείας σε ένα τιμολόγιο.

1. Σε προσχέδιο τιμολογίου, στο υποπλέγμα **Προπληρωμές και τις Προκαταβολές** επιλέξτε **Δημιουργία** για να δημιουργήσετε μια νέα προκαταβολή ή μια προπληρωμή. 
2. Στη σελίδα **Γρήγορη δημιουργία**, προσθέστε τις απαραίτητες πληροφορίες και, στη συνέχεια, επιλέξτε **Αποθήκευση**. Η προκαταβολή ή η προπληρωμή δημιουργείται στη σύμβαση έργου που σχετίζεται με το τιμολόγιο. Η προκαταβολή ή η προπληρωμή επισημαίνεται αυτόματα ως **Έτοιμο για τιμολόγηση** και μετά προστίθεται στο υποπλέγμα **Προπληρωμές και Προκαταβολές** στη σελίδα **Τιμολόγιο**.

## <a name="reconcile-an-invoiced-retainer-or-advance"></a>Συμφωνία μιας τιμολογημένης προκαταβολής ή προπληρωμής

Μετά την τιμολόγηση μιας προκαταβολής ή μιας προπληρωμής, μπορούν να χρησιμοποιηθούν ή να συμφωνήσουν σε ένα τιμολόγιο με χρόνο, έξοδα, ορόσημα ή άλλες χρεώσεις βάσει έργου. Ο πελάτης θα βλέπει το ποσό τιμολογίου μειωμένο από το ποσό προκαταβολής ή προπληρωμής που χρησιμοποιείται σε αυτό το τιμολόγιο.

Σε κάθε τιμολόγιο που δημιουργείται για μια σύμβαση έργου που έχει τιμολογημένες προκαταβολές ή προπληρωμές, το Project Operation εφαρμόζει αυτόματα την προκαταβολή ή την προπληρωμή στο τιμολόγιο.

Αυτό μπορείτε να το δείτε στο πλέγμα **Προκαταβολές και προπληρωμές που εφαρμόζονται** στη σελίδα **Τιμολόγιο**. Στον ακόλουθο πίνακα παρέχονται πληροφορίες σχετικά με τα πεδία στο πλέγμα **Προκαταβολές και προπληρωμές που εφαρμόζονται** στη σελίδα **Τιμολόγιο έργου**.
