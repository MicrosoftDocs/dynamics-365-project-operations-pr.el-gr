# <a name="whats-new-december-2020---project-operations-lite-deployment---deal-to-proforma-invoicing"></a>Τι νέο υπάρχει για τον Δεκέμβριος 2020 - lite ανάπτυξη Project Operations - συμφωνία για προτιμολόγηση

_**Ισχύει για:** Ελαφριά ανάπτυξη - συμφωνία για προτιμολόγηση_

Αυτό το θέμα ισχύει για τα ακόλουθα στοιχεία και τις εκδόσεις του Dynamics 365 Project Operations:

  - Project Operations σε περιβάλλον Dataverse έκδοση 4.5.0.134 

Ο παρακάτω πίνακας παραθέτει τις ενημερώσεις στο Project Operations σε περιβάλλον Dataverse έκδοση 4.5.0.134.

| --- | --- | --- |
| Χρέωση και τιμολόγηση | 1986009 | Οι γραμμές ημερολογίου που δημιουργήθηκαν με μη αυτόματο τρόπο έχουν ασυνεπή απόδοση όταν επιβεβαιωθεί πριν τη σύνδεση του έργου ή την αποσύνδεσή του από μια γραμμή σύμβασης. |
| Χρέωση και τιμολόγηση | 2014153 | Τα προϊόντα δεν τιμολογούνται όταν εκτελούνται από το χρονοδιάγραμμα τιμολόγησης. |
| Χρέωση και τιμολόγηση | 2014159 | Τα προϊόντα δεν έλκονται όταν το τιμολόγιο ανανεώνεται για συναλλαγές. |
| Χρέωση και τιμολόγηση | 2028174 | Οι ενημερώσεις για τις λεπτομέρειες της γραμμής τιμολογίων πρέπει να ακολουθούν τη λογική του ημερολογίου διόρθωσης. |
| Χρέωση και τιμολόγηση | 2028315 | Επιδιορθώσεις συμπεριφοράς ένθετου πλέγματος με δυνατότητα επεξεργασίας. |
| Χρέωση και τιμολόγηση | 2031070 | Η προσαρμογή λεπτομερειών γραμμής διορθωτικών τιμολογίων πρέπει να ακολουθεί τη λογική του ημερολογίου διόρθωσης. |

| Χρέωση και τιμολόγηση | 2057874 | Σωστή σύνδεση συναλλαγής που έχει δημιουργηθεί κατά τη διάρκεια επεξεργασίας λεπτομερειών γραμμής τιμολογίων. |
| Χρέωση και τιμολόγηση | 2057875 | Σωστή προέλευση συναλλαγής που έχει δημιουργηθεί κατά τη διάρκεια επεξεργασίας λεπτομερειών γραμμής τιμολογίων. |
| Χρέωση και τιμολόγηση | 2057944 | Κατάσταση για μη υπέρβαση που έχει οριστεί ως «Δεσμευμένη» για πραγματικά στοιχεία που δεν είναι έτοιμα για τιμολόγηση από μια διόρθωση τιμολογίου. |
| Χρέωση και τιμολόγηση | 2057963 | Διαίρεση προνομίου Create\_Product από προνόμιο Create\_ProjectContract. |

| Χρέωση και τιμολόγηση | 2067624 | Το ποσό που έχει συμβληθεί πρέπει να επισημανθεί ως «Συνιστώμενη εργασία» κατά τη δημιουργία ορόσημων. |
| Χρέωση και τιμολόγηση | 2086880 | Απόκρυψη κουμπιού **Πρόταση** στην κορδέλα για γραμμές προσφοράς βάσει έργου. |

|   Διαχείριση ευκαιριών | 2022204 | Το πλέγμα χρέωσης βάσει έργου στην καρτέλα **Τιμολόγηση εργασιών** στη σελίδα **Έργο** θα πρέπει να αποκρύπτει το πλέγμα βάσει έργου εάν δεν υπάρχει γραμμή προσφοράς/σύμβασης με **Όλες τις εργασίες** και αντίστροφα. |
|   Διαχείριση ευκαιριών | 2086601 | Οι ρόλοι και οι κατηγορίες που απενεργοποιούνται δεν πρέπει να αντιγράφονται στη λίστα χρεώσιμων ρόλων και χρεώσιμων κατηγοριών στις γραμμές προσφοράς και τις γραμμές σύμβασης. |
| Προγραμματισμός και παρακολούθηση έργων | 1949065 | Η ορατότητα δεδομένων λειτουργεί σωστά σε μεγέθυνση 200% |

| Προγραμματισμός και παρακολούθηση έργων | 2057197 | Η εκτίμηση του αντιγράφου γραμμής με αναφορά εργασίας δεν υποστηρίζεται. |
| Προγραμματισμός και παρακολούθηση έργων | 2060687 | Η προειδοποίηση ζώνης ώρας εξαφανίζεται πλέον μετά από μια συγκεκριμένη διάρκεια. |

| Χρόνος και έξοδα | 2034882 | Το κουμπί **Νέο** εμφανίζεται δύο φορές στη γραμμή εντολών για καταχωρήσεις λωρας όταν είναι εγκατεστημένο το Dynamics 365 Field Service. |
