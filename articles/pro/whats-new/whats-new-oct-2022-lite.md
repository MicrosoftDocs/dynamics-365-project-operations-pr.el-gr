# <a name="whats-new-october-2022---project-operations-lite-deployment"></a>Τι νέα υπάρχουν, Οκτώβριος 2022 - ελαφριά ανάπτυξη Λειτουργιών Έργου

_**Ισχύει για:** Ελαφριά ανάπτυξη - συμφωνία για προτιμολόγηση_

Αυτό το άρθρο ισχύει για τα ακόλουθα στοιχεία και εκδόσεις του Microsoft Dynamics 365 Project Operations:

- Project Operations στο περιβάλλον Dataverse έκδοση 4.57.0.176

## <a name="features-included-in-this-release"></a>Δυνατότητες που περιλαμβάνονται σε αυτήν την έκδοση

| Περιοχή δυνατοτήτων | Όνομα δυνατότητας | Περισσότερες πληροφορίες |
| --- | --- | --- |
| Προγραμματισμός και παρακολούθηση έργων | **Εξωτερικός προγραμματισμός του Project Operations**<br>Η λειτουργία εξωτερικού προγραμματισμού επιτρέπει στους πελάτες να δημιουργούν, να ενημερώνουν και να διαγράφουν εγγενώς οντότητες που σχετίζονται με δομές ανάλυσης εργασίας (WBS), χρησιμοποιώντας τα εγγενή API του Dataverse, χωρίς τα τρέχοντα όρια που επιβάλλονται από το Project for the Web. | [Εξωτερικός προγραμματισμός](/dynamics365/project-operations/project-management/external-scheduling) |
| Ανάπτυιξη και ρύθμιση παραμέτρων | <p>**Να επιτρέπεται στους πελάτες να επιλέγουν τον τύπο ανάπτυξης**<br>Οι ενημερώσεις βάσει προϊόντος (ΚΙΝΑΛ) του Project Operations χρησιμοποιούνται για την αυτόματη εγκατάσταση της λύσης διπλής εγγραφής του Project Operations όταν οι λύσεις πυρήνα διπλής εγγραφής και ενορχήστρωσης εγκαθίστανται στο περιβάλλον.</p><p>Αυτή η δυνατότητα παρουσιάζει μια νέα παράμετρο **Συμπεριφοράς αναβάθμισης λύσης** στις παραμέτρους έργου. Όταν αυτή η παράμετρος ορίζεται σε **Μόνο Lite**, τα PUD δεν θα εγκαθιστούν πλέον αυτόματα τη λύση διπλής εγγραφής του Project Operations, ακόμα κι εάν οι λύσεις πυρήνα διπλής εγγραφής και ενορχήστρωσης εγκαθίστανται στο περιβάλλον. Αυτή η συμπεριφορά θα ωφελήσει τους πελάτες που σχεδιάζουν να χρησιμοποιήσουν λύσεις διπλής εγγραφής για την ενσωμάτωση παραγγελιών πωλήσεων σε εφαρμογές οικονομικών και επιχειρηματικών δραστηριοτήτων, αλλά θέλουν να χρησιμοποιούν το Project Operations σε λειτουργία Lite (δηλαδή χωρίς ενοποίηση σε εφαρμογές οικονομικών και επιχειρηματικών δραστηριοτήτων).</p> | |
