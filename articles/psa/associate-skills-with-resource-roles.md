<span data-ttu-id="da65c-104">Ενώ προσθέτατε ρόλους πόρων, πιθανώς να συσχετίσατε ήδη ορισμένες δεξιότητες σε αυτούς.</span><span class="sxs-lookup"><span data-stu-id="da65c-104">While you were adding resource roles, you probably already associated some skills to them.</span></span> <span data-ttu-id="da65c-105">Κατά καιρούς, ίσως χρειαστεί να προσθέσετε περισσότερες δεξιότητες σε έναν ρόλο πόρων.</span><span class="sxs-lookup"><span data-stu-id="da65c-105">From time to time, you might need to add additional skills to a resource role.</span></span>
